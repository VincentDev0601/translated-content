{{ApiRef("DOM")}}

语法

Object.queryCommandState(String command)

描述

返回 指定命令 在对象内的状态码（1表示指定命令在对象内已执行；0表示指定命令在对象内未执行，处于可执行状态；-1表示指定命令在对象内处于不可用状态）
