exitPointerLock 方法可异步的解锁鼠标（通过{{domxref("Element.requestPointerLock")}}锁定的）。

追踪是否解锁成功，需要监听{{event("pointerlockchange")}} 和{{event("pointerlockerror")}} 事件。

语法

document.exitPointerLock();
