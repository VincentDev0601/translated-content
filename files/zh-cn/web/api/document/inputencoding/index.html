概述

返回一个字符串,代表当前文档渲染时所使用的编码.(比如utf-8).

警告: 不要再使用该属性.该属性在DOM 4 规范(草案)中已经被废弃. Gecko 在未来的版本中将会删除它.

语法

encoding = document.inputEncoding;

inputEncoding 是个只读属性.

规范
