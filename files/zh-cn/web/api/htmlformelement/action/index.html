HTMLFormElement.action 这个js语句能够定义一个{{HTMLElement("form")}}元素中的action属性

表单的action属性是在服务器上提交表单，这个属性可以被检索或者设置。

语法

string = form.action
form.action = string

例子

form.action = "/cgi-bin/publish";

详细说明

HTML 5, Section 4.10.19.6, Form submission
