作为ServiceWorkerRegistration的一部分定义，会返回该ServiceWorker的序列化脚本url. 必须与document注册ServiceWorker的地址同源.

语法

someURL = ServiceWorker.scriptURL

值

一个 {{domxref("USVString")}} (另见 WebIDL definition of USVString.)
