{{APIRef("DOM Events")}}

 

WheelEvent.deltaZ 只读属性是一个 double 类型值, 声明 Z 轴滚动量以WheelEvent.deltaMode 为单位.

 

语法

var dZ = event.deltaZ;

例子

var syntheticEvent = new WheelEvent("syntheticWheel", {"deltaZ": 4, "deltaMode": 0});

console.log(syntheticEvent.deltaZ);
