{{APIRef("DOM")}}

DOMString 是一个 UTF-16 字符串。由于 JavaScript 已经使用了这样的字符串，所以 DOMString 直接映射到一个 {{jsxref("String")}}。

将 null传递给接受 DOMString 的方法或参数时通常会把其 stringifies 为“null”。
