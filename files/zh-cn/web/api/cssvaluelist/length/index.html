{{APIRef("DOM")}}

{{domxref("CSSValueList")}} 接口的只读属性 length 表示列表中的 {{domxref("CSSValue")}} 数量。有效的索引区间为 0 到 length-1。

语法

var length = cssValueList.length;

值

一个 unsigned long 数值，表示此列表中 {{domxref("CSSValue")}} 的个数。
