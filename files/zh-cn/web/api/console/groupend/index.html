{{APIRef("Console API")}}

在 Web控制台中退出一格缩进(结束分组). 请参阅 {{domxref("console")}} 中的Using groups in the console 来获取它的用法和示例.

{{AvailableInWorkers}}

语法

console.groupEnd();

参数

None.
