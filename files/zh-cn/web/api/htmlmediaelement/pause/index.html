{{APIRef("HTML DOM")}}

HTMLMediaElement.pause() 方法可用来暂停媒体的播放，如果媒体已经处于暂停状态，该方法没有效果.

语法

HTMLMediaElement.pause()

参数

无.

返回值

无.

异常

无.
