{{APIRef("HTML DOM")}}

HTMLMediaElement.controls 这个负责控制html中的controlsHTML属性, 它负责控制播放的媒体(视频或音频)的控制条是否显示。.

语法

var ctrls = video.controls;
audio.controls = true;

返回值

类型 {{domxref("Boolean")}}. 是否成功

例子

var obj = document.createElement('video');
obj.controls = true;
