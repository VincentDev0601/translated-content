{{APIRef("Navigation Timing")}}

概要

只读属性 Performance.navigation 会返回一个 {{domxref("PerformanceNavigation")}} 对象。这个对象表示出现在当前浏览上下文的 navigation 类型，比如获取某个资源所需要的重定向次数。

语法

navObject = performance.navigation;
