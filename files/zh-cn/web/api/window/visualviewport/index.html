{{domxref("Window")}} 接口的visualViewport只读属性返回一个{{domxref("VisualViewport")}}对象，该对象表示给定窗口的可视视口。

语法

var visualViewport = Window.visualViewport

值

一个 {{domxref("VisualViewport")}} 对象。
