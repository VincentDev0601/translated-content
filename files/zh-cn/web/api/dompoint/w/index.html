{{APIRef("DOM")}}

DOMPoint 的 w 属性表示该点的空间透视值。

语法

var perspective = DOMPoint.w;

值

双精度浮点值，表示该点的空间透视值。这个值的类型并没有严格限制，意味着它可以是 {{jsxref("NaN")}} 或 {{jsxref("Infinity", "±Infinity")}}。 默认值为 1.0。
