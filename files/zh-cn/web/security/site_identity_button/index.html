 Site Identity Button 是火狐的一个特性，用于向用户提示更多他们已经访问过的网站的信息。

这个特性的使用取决于你网站的配置，按钮会显示许多不同的图标。

如果站点身份按钮没有按照您的配置显示出来的话（如你想要看到一个绿色的锁却显示一个橙色的三角图标），你可以打开火狐开发者工具的控制台，在控制台里能找到一些原因：

  1. 确定您的控制台在'security'目录下显示信息。
  2. 强制刷新有问题的站点
  3. 耐心等待可能会出现的一些安全信息

由于以下的原因之一，UI的安全等级将会降低：

  • mixed content - 页面是经过TLS传输，但是页面没有经过subresources加载；这种情况下，在控制台里会看到‘mixed content’的信息提示。
  • Use of a weak cipher - 当使用TLS时而强密码是不可用的，会显示一些关于弱密码的信息提示（如RC4）
  • Insecure renegotation - 老版本的TLS在设计上有一个缺陷；如果您的服务在使用低版本的TLS，UI的等级将会降低且控制台不会有任何信息提示。
