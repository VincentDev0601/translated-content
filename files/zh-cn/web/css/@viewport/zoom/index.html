{{ CSSRef }}

zoom CSS 属性会根据 {{cssxref("@viewport")}} 来初始化一个缩放因数。

当设置1.0 或 100%时表示不缩放。更大的值放大，更小的值缩小。

{{cssinfo}}

语法

/* Keyword value */
zoom: auto;

/* <number> values */
zoom: 0.8;
zoom: 2.0;

/* <percentage> values */
zoom: 150%;

值

auto
根据`viewport`来既定当前标签的缩放。
<number>
必须是一个非负数。1表示没有缩放，大于1表示放大的倍数，小于1亦然。
<percentage>
必须是一个非负的百分比。以100%为基础进行缩放。
