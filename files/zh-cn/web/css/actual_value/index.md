<h2 id="概述">概述</h2>

<p>一个 CSS 属性的<strong>实际值（actual value）</strong>是应用值（used value）被应用后的近似值。例如，一个用户代理可能只能渲染一个整数像素值的边框（实际值），并且该值可能被强制近似于边框的计算宽度值。</p>
