Object.eval() 方法用于在对象的上下文中对 JavaScript 代码字符串求值，但该方法已被移除。

语法

obj.eval(string)

参数

string
包含任意 JavaScript 表达式、语句或一组语句的字符串。表达式中可包含已有对象的变量与属性。

描述

eval 方法已从对象方法中移除。可使用全局 {{jsxref("Global_Objects/eval", "eval()")}} 函数替代该方法。
