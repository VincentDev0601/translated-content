Object.getNotifer() 方法用于创建可人工触发 change 事件的对象，但该方法在浏览器中已被废弃。

语法

Object.getNotifier(obj)

参数

obj
获取通知的对象。

返回值

与传入对象相关联的通知对象。

描述

通知对象可触发 Object.observe() 所观察到的人工变动。

规范

Strawman proposal specification.
