{{JSRef}} {{obsolete_header("2")}}

 __count__ 属性曾经用来存放对象的可枚举的属性的个数，但是已经被废除。

语法

obj.__count__

示例

{ 1: 1 }.__count__              // 1
[].__count__                    // 0
[1].__count__                   // 1
[1, /* hole */, 2, 3].__count__ // 3

详细说明

无需详细说明
