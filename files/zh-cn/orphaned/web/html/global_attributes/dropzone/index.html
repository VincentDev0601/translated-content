dropzone 全局属性 是个枚举属性，表示什么内容类型可以拖放到元素上，使用 Drag and Drop API，它可以拥有以下值：

  • copy，它表明拖放行为会创建被拖放元素的副本。
  • move，它表明被拖放元素会移动到新的位置。
  • link，它会创建被拖放数据的链接。
