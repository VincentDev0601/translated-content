这个页面列出了 Firefox 开发者工具的所有快捷键使用方法 。

第一部分列出打开每个工具的快捷键，第二部分列出适用于工具箱本身的快捷键，之后为每个工具部分的快捷键。

依赖本地环境的快捷键没有记录在这个页面。

打开和关闭工具

这些快捷键可以在浏览器窗口打开指定的工具。对于工具箱中集成的工具，如果已被打开，也可以通过这些快捷键进行关闭操作。对于在新窗口打开的工具（如浏览器控制台），你必须关闭窗口来关闭工具。

打开工具箱（激活最近使用的工具） Ctrl + Shift + I Cmd + Opt + I Ctrl + Shift + I
唤起工具箱到前台（当工具箱为分离窗口模式且没有在前台运行） Ctrl + Shift + I 或 F12 Cmd + Opt + I 或 F12 Ctrl + Shift + I 或 F12
关闭工具箱（当工具箱为分离窗口模式且在前台运行） Ctrl + Shift + I 或 F12 Cmd + Opt + I 或 F12 Ctrl + Shift + I 或 F12
打开 Web 控制台1 Ctrl + Shift + K Cmd + Opt + K Ctrl + Shift + K
切换工具箱 Ctrl + Shift + I Cmd + Opt + I Ctrl + Shift + I
打开调试器2 Ctrl + Shift + S Cmd + Opt + S Ctrl + Shift + S
打开样式编辑器 Shift + F7 Shift + F71 Shift + F7
打开性能分析器 Shift + F5 Shift + F51 Shift + F5
打开网络监视器3 Ctrl + Shift + Q Cmd + Opt + Q Ctrl + Shift + Q
切换开发者工具栏 Shift + F2 Shift + F21 Shift + F2
切换响应式设计视图 Ctrl + Shift + M Cmd + Opt + M Ctrl + Shift + M
打开浏览器控制台4 Ctrl + Shift + J Cmd + Shift + J Ctrl + Shift + J
打开浏览器工具箱 Ctrl + Alt +Shift + I Cmd + Opt +Shift + I Ctrl + Alt +Shift + I
打开代码草稿纸 Shift + F4 Shift + F4 Shift + F4
打开 WebIDE Shift + F8 Shift + F8 Shift + F8
存储探查器5 Shift + F9 Shift + F9 Shift + F9

1.不同于其他工具箱内工具，该快捷键无法关闭 Web 控制台。因为把焦点集中到了 Web 控制台的命令行。关闭 Web 控制台可以使用全局工具箱快捷键 Ctrl+Shift+I （Mac：Cmd+Opt+I）。

2.从 Firefox 66 开始，这个快捷键不再关联调试器。

3.在 Firefox 55 以前的版本上，这个快捷键是 Ctrl+Shift+Q （Mac：Cmd+Opt+Q）

4.到 Firefox 38 为止的版本上，当浏览器控制台在正常 Firefox 窗口上被隐藏，相同的组合键为关闭浏览器控制台。从 Firefox 38 起，如果当浏览器控制台在正常 Firefox 窗口上被隐藏，上面这个组合键将把焦点定位到浏览器控制台。

5.该工具在默认情况下是禁用的，需要在设置面板中启用后才能运作。

代码编辑器

此表列出了代码编辑器的默认快捷键。

在开发工具选项的编辑器偏好中，您可以选择使用 Vim、Emacs 或 Sublime Text 的快捷键。

选择这些,访问 about:config，选择设定 devtools.editor.keymap 的值为 "vim" 或 "emacs"或 "sublime"。通过这个设定。通过该方法你可以设定开发者工具中所有的代码编辑器使用的快捷键。并且你需要重新启动编辑器更改才能生效。

从 Firefox 33 起，更改编辑器快捷键可以通过开发者工具箱选项中的编辑器偏好实现。而不需要通过 about:config。
