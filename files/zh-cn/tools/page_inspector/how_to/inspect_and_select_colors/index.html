在CSS面板中的Rules view规则视图，如果样式规则包含颜色值，颜色值的旁边会显示一个颜色样板。

点击颜色样板，将弹出选择面板，可以通过选择面板修改当前颜色的值。

The color picker includes an eyedropper icon: clicking this icon enables you to use the eyedropper to select a new color for the element from the page:

{{EmbedYouTube("0Zx1TN21QOo")}}

Starting in Firefox 40, clicking the color sample while holding down the Shift key will change the color format:
