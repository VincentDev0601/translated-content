ui.tooltipDelay 存储着一个单位为毫秒的时间数值,该数值表示鼠标停留在某个元素上多长时间后显示出提示信息.

  • 类型:integer
  • 默认值:500
  • 默认已存在: no
  • 支持版本: Gecko 11.0 {{ geckoRelease("11.0") }}
  • 状态: Active; last updated 2012-02-21
  • 介绍: Pushed to Nightly on 2011-12-15
  • Bugs: {{ bug("204786") }}

值

整数(单位为毫秒, 默认值: 500)
该数值表示鼠标停留在某个元素上多长时间后显示出提示信息,单位为毫秒,默认值为500ms.
