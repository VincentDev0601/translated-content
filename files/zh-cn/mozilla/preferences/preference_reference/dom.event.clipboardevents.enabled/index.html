dom.event.clipboardevents.enabled 控制,当用户在网页上执行复制,粘贴或者剪切动作时,是否触发 oncopy, oncut 和 onpaste 事件.

  • 类型:boolean
  • 默认值:true
  • 默认已存在: no
  • 支持版本: Gecko 13.0 {{ geckoRelease("13.0") }}
  • 状态: Active; last updated 2012-02-15
  • 介绍: Pushed to Nightly on 2012-02-14
  • Bugs: {{ bug("542938") }}

值

true (默认)
可以触发 oncopy, oncut 和 onpaste 事件
false
不会触发 oncopy, oncut 和 onpaste 事件
