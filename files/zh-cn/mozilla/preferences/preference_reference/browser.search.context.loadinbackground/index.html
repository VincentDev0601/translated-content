browser.search.context.loadInBackground 控制右键菜单中的"通过<搜索引擎>搜索<选中文字>"打开新标签页后,立即激活该标签页,还是保持当前标签页状态不变.

  • 类型:boolean
  • 默认值:false
  • 默认已存在: yes
  • 支持版本: Firefox 13.0
  • 状态: Active; last updated 2012-02-17
  • 介绍: Pushed to Nightly on 2012-02-15
  • Bugs: {{ bug("727131") }}

值

true
搜索页面在后台打开,保持当前标签页不变
false (默认)
搜索页面在前台打开(打开并激活).
