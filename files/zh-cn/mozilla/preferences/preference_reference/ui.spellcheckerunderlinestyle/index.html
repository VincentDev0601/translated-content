  • 类型:integer
  • 默认值:5
  • 默认已存在: no
  • 支持版本: Gecko 1.9.2 {{ geckoRelease("1.9.2") }}
  • 状态: Active; last updated 2012-02-22
  • 介绍: Pushed to Nightly on 2009-04-03

值

0

1
点状线
2

3

4
双实线
5 (默认值)
