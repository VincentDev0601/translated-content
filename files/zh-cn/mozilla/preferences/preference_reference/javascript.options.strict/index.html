当代码执行时,如果这条语句没有发生运行时错误,但却是用了非标准的特性,写法有问题,可能导致隐含的逻辑错误,则会生成JavaScript警告信息.该选项就控制着这些警告信息是否要打印到JavaScript控制台上.

 

可用的值和对应的效果

True: 在错误控制台上显示JavaScript错误和警告.

False: 在错误控制台上只显示JavaScript错误.(默认值)
