Mozilla Toolkit 是建立在Gecko上的应用程序接口集(APIs)，用来提供对XUL应用的高级服务. 这些服务包含:

  • Profile 管理
  • Chrome 注册
  • 浏览历史
  • 扩展和主题管理
  • 应用更新服务
  • 安全模式
