collapsed
Type: boolean
类型：布尔类型(boolean)
If true, then the element is collapsed and does not appear. It is equivalent to setting the CSS visibility property to collapse.
如果是真，那么这个元素可以折叠并且隐藏。相当于设置CSS的 visibility 属性为 collapse。
