概要

BluetoothStatusChangedEvent API可以访问有关蓝牙设备状态的任何更改的信息。

当触发以下某个事件时，会发生状态更改:

  • {{Event("a2dpstatuschange")}} : 当A2DP连接状态发生变化时。看到{{domxref("BluetoothAdapter.ona2dpstatuschanged")}} 了解更多信息。
  • {{Event("hfpstatuschange")}} : 发生在HFP连接状态发生变化时。看到{{domxref("BluetoothAdapter.onhfpstatuschanged")}} 了解更多信息.
  • {{Event("pairedstatuschange")}} : 当设备的配对状态发生变化时。看到{{domxref("BluetoothAdapter.onpairedstatuschanged")}} for more information.了解更多信息
  • {{Event("scostatuschange")}} : 它发生在SCO连接状态改变时。看到{{domxref("BluetoothAdapter.onscostatuschanged")}} for more information.

接口概述

interface BluetoothStatusChangedEvent: Event
{
  readonly attribute DOMString address;
  readonly attribute boolean status;
};

属性

{{domxref("BluetoothStatusChangedEvent.address")}} {{readonlyinline}}
表示蓝牙微网中状态发生变化的设备地址的字符串。.
{{domxref("BluetoothStatusChangedEvent.status")}} {{readonlyinline}}
表示连接当前状态的布尔值。它可以被启用（true）或禁用（false）。

方法

目前没有。

规范
