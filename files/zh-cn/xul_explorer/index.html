XUL Explorer是一款提供一个很容易上手来进行XUL测试的 XULRunner 应用。这个编辑器很简单，他能在线预览或者在一个独立窗口预览XUL。它有一个列表的代码片段(小片段的XUL或JavaScript),可以快速插入到编辑器中。XUL可以加载和保存到文件。其中的XUL验证器和 错误控制台 可以帮助您找到问题所在。帮助菜单提供了访问 MDC上XUL信息的通道，更有 “keyword” 查找框帮助查找XUL元素。

蓝图

特性:

  • 预览窗格可以切换与编辑。预览在弹出的窗口中仍然是可用的。

  • 可以设置编辑器改变的内容在预览中自动动态更新。

  • 当状态栏中的指示器显示错误时， 双击它将显示错误控制台。
  • 可以设置是否制定在启动时编辑器加载某个代码段。
  • 可以设置是否包含一段用户的代码段文件，在启动时，这些代码段将与编译中的代码段进行合并。
  • 允许双击或者拖拉插入代码段。
  • File > New 允许用户选择任何“模板”的片段，以及空白。
  • Simple XUL Checker 能够用于 ‘验证’ XUL 代码段. 它检查元素和属性的名称是否合法，包括一个给定的元素属性是否合法。
  • 支持检查属性值是否合理 (布尔型和枚举值) - XUL Checker

  • 支持“最佳实践”的检查，如：命令和键值的使用，DTD中的字符串等等- XUL Checker

  • 多标签编辑器支持

  • 支持向导生成通用项目扩展

  • 支持使用Firefox扩展测试模式测试扩展。
  • Venkman 支持
  • DOM Inspector支持

特性:

  • 支持更多的”最佳实践“检查，例如：更多DTD文件中 A11y检查，字符串等等 - XUL Checker

  • 允许用户自由添加代码段

  • 增加更加边栏来显示更多的功能- Property Sidebar and Project Sidebar
  • 支持同一代码生成 XUL files, JS XPCOM, 和 XULRunner applications
