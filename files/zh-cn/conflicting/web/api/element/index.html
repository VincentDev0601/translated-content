{{APIRef("Shadow DOM")}}

Slotable mixin 定义了允许节点成为 {{htmlelement("slot")}} 元素的内容的特性——下面的特性被包含在 {{domxref("Element")}} 和 {{domxref("Text")}} API 之中。

属性

{{domxref("Slotable.assignedSlot")}} {{readonlyInline}}
返回节点所插入的 {{htmlelement("slot")}} 元素。

方法

无。
