{{ Non-standard_header() }}

简介

这是Microsoft Internet Explorer专有的用于替代标准的 {{domxref("EventTarget.removeEventListener()")}} 的方法。

语法

target.detachEvent(eventNameWithOn, callback)
target
将要移除事件的DOM节点
eventNameWithOn
将要移除的事件名，以“on”为前缀（例如它是一个事件处理程序）。 例如，您可以使用“onclick”移除点击事件的事件处理程序。
callback
注销事件后的回调函数
