次のプログラムが、NSPRやその一部を使っていると知られています。

  • Gecko を使ったプログラム (Mozilla Application Suite, Firefox, Thunderbird, Camino, など)
  • 多くの Fedora/Red Hat と Sun のサーバ・アプリケーション。 これらのほとんどは Netscape serversが由来です。
  • Netscape の同窓生が彼らの新しい企業でNSPRを採用しています: Good Technology, Kontiki, Tellme Networks
