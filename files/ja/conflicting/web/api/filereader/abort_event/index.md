**`FileReader.onabort`** プロパティには、[`abort`](/ja/docs/Web/Events/abort) イベントが発生したとき、つまりファイルの読み取り処理が中止されたときに実行されるイベント ハンドラが含まれています。

## シンタックス

```js
reader.onabort = function() { ... };
```
