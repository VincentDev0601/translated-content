{{ApiRef}}

概要

16 進でエンコードされた値（例えば、クッキー）をデコードします。

構文

regular = window.unescape(escaped)
  • regular : 通常の文字列
  • encoded : エンコードされた文字列

例

alert( unescape("%5C") );  // 表示結果: "\"

alert( unescape("https%3A//developer.mozilla.org") );  // 表示結果: "https://developer.mozilla.org"

仕様

{{DOM0}} 但し、ECMA-262 の非標準化セクションで言及されています。
