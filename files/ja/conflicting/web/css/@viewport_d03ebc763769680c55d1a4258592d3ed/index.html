{{CSSRef}}

min-zoom は CSS の記述子で、 {{cssxref("@viewport")}} アット規則で定義され、文書の最大表示倍率を設定します。ブラウザーは自動的にもユーザーのリクエストでも、これより拡大することができません。

表示倍率 が 1.0 または 100% が、拡大縮小なしに対応します。より大きい値は拡大、より小さい値は縮小です。

構文

/* キーワード値 */
max-zoom: auto;

/* <number> 値 */
max-zoom: 0.8;
max-zoom: 2.0;

/* <percentage> 値 */
max-zoom: 150%;

値

auto
{{glossary("user agent", "ユーザーエージェント")}}が文書の表示倍率の上限を設定します。
{{cssxref("<number>")}}
非負の数値で、表示倍率の上限です。
{{cssxref("<percentage>")}}
非負のパーセント値で、表示倍率の上限です。
