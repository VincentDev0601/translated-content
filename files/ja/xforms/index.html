XForms は W3C によって設計された次世代オンライン・フォームです。XML Schema や XPath、XML Events のような他の W3C 標準を利用して、XForms は現在の HTML フォーム・モデルのいくつかの制限を解決しようとします。他の XForms がもたらす強みは、プレゼンテーションからのデータの分離と強力なデータタイプ、名前/値の組の代わりにサーバに XML データを提出する能力、さまざまなデバイスにそれらを表示できるようにフォームを作成する説明的な方法です。XForms は W3C の仕様です。
