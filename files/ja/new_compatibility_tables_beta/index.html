"新しい互換性の表はベータ版です"というリンクから来ましたか?（違いますか?新しい互換性の表を見たい場合はのベータテスターになってください。）

テストしていただきありがとうございます。これらは大きなプロジェクトの一部に過ぎません。私たちはすべてのブラウザ互換性データを構造化 JSON に移行しています。

データが新しい形式に変換されると、表も新しくなります。

編集

互換性データは browser-compat-data GitHub repository の JSON ファイルに移動しました。

互換性データに貢献したい場合はプルリクエストを送るか、イシューを立ててください。

助けるには

データに問題があれば GitHub リポジトリにイシューを立ててください。

表の見た目や機能に問題がある場合は、表の上にあるドロップダウンメニューから「問題を報告」ボタンを押してください。

時間が許せばアンケートに参加していただけると嬉しいです。
