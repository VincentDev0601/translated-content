IP アドレスはネットワークに接続された、インターネットプロトコル(IP) を使うすべての端末に割り当てられる番号です。

"IP アドレス" は IPv6 がもっと広がっていくまでは、典型的にはまだ 32-ビット IPv4 アドレスを参照します。
