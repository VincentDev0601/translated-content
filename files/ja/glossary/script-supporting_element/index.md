<p><span class="seoSummary">{{Glossary("HTML")}} 文書において、<strong>スクリプト対応要素</strong>はページの表示やレイアウトに直接貢献しない要素です。その代わり、スクリプト自体やスクリプトからのみ使用される情報を持っています。</span>これらの要素は重要な場合もありますが、ページのスクリプトが正確に作用しなければ表示されているページに影響しません。</p>

<p>スクリプト対応要素は {{HTMLElement("script")}} 及び {{HTMLElement("template")}} の2つしかありません。</p>
