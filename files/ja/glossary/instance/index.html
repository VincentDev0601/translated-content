{{glossary("constructor", "コンストラクタ")}} によって作成された{{glossary("object","オブジェクト")}}は、そのコンストラクタのインスタンス（実体）と呼ばれます。

var a = function(){};
//a is constructor.

var b = new a();
//b is instance.

{{訳注("JavaScript は、 Java のようなクラスベースの言語とは違い、new を用いたクラスのインスタンス化だけではなく、オブジェクト初期化子（{}）を用いても、オブジェクトを作成できます。")}}
