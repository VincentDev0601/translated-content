--- title: Pseudo-element slug: Glossary/Pseudo-element tags: - CSS - CodingScripting - Glossary translation_of: Glossary/Pseudo-element ---

CSS では擬似要素セレクターを使うことで、選択すべき特定の HTML 要素がない状況でも、ドキュメントのコンテンツの一部に、スタイルを適用できます。例えば、各段落の最初の文字を、それぞれの要素の中に置いていくのではなく、p{{ Cssxref("::first-letter") }} で、それらのすべてに、スタイルを指定することができます。
