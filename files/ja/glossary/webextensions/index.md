<p>WebExtensions はFirefoxでブラウザー拡張機能を開発するためのクロスブラウザーシステムです。このシステムには APIが提供され、これはかなりにわたって様々なブラウザー、例えば Mozilla Firefox, Google Chrome, Opera Browser, Microsoft Edge をサポートします。</p>
