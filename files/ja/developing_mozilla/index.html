ツール

  • Bugzilla : Mozilla プロジェクトのためのバグデータベース。
  • MXR (formerly LXR): ソースリポジトリのブラウズと検索。
  • Bonsai : 誰が、どのファイルをいつ変更したのかを見つける。
  • Tinderbox : tree ステータスを表示。チェックアウト前とチェックイン前に確認してください。
  • クラッシュの追跡: Socorro, Talkback
  • パフォーマンス追跡のための graphs.mozilla.org。
