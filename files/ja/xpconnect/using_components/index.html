XPConnect は、 Mozilla および xpcshell で透過的に動作し、 XPCOM コンポーネントへのアクセスを提供します。

通常、スクリプトは次のように開始します:

var Cc = Components.classes;
var Ci = Components.interfaces;

コンポーネントを取得したい場合は、次のようにします:

var rc = Cc["@mozilla.org/registry;1"];
var rs = rc.getService(Ci.nsIRegistry);

関連項目:

  • xpcshell -- JavaScript へのコマンドラインインターフェイスを取得する方法

より詳しい情報

既に述べたように、次のようなアドオンスクリプトを開始するのが一般的です:

var Cc = Components.classes;
var Ci = Components.interfaces;

開始する別の方法もあります。これは上記と全く同じです.

var {Cc: classes, Ci: interfaces} = Components;

クラスとインターフェイスだけではありません。

var {
    Cu: utils,
    Ci: interfaces,
    Cc: classes,
    Cr: results,
    Cs: stack,
    Cm: manager,
    Ce: Exception,
} = Components;
