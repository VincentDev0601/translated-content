-moz-tree-row CSS 擬似要素は、列の選択と tree 列のスタイル付けに使用されます。

関連付けられた要素

  • {{XULElem("treerow")}}

構文

treechildren::-moz-tree-row { style プロパティ }

Style プロパティ

  • {{cssxref("background")}}
  • {{cssxref("border")}}
  • {{cssxref("margin")}}
  • {{cssxref("outline")}}
  • {{cssxref("padding")}}
  • {{cssxref("display")}}
  • {{cssxref("-moz-appearance")}}

例

treechildren::-moz-tree-row( foo bar )
{
    margin: 2%;
}

...次の位置に適用されます...

  <treerow properties="foo">...</treerow>
