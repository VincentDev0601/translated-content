列挙型 JSType の値は、 JavaScript の値の型を表します。

値 説明
JSTYPE_VOID undefined 値
JSTYPE_OBJECT JavaScript オブジェクト
JSTYPE_FUNCTION 関数
JSTYPE_STRING 文字列
JSTYPE_NUMBER 数値
JSTYPE_BOOLEAN 真偽値 (true / false)
JSTYPE_NULL null 値
JSTYPE_XML XML オブジェクト
