C 言語 / C++ の整数型のエイリアスです。

構文

typedef ... jsint;
typedef ... jsuint;

typedef ... jsword;
typedef ... jsuword;

typedef ... intN;
typedef ... uintN;

typedef ... int8;
typedef ... uint8;
typedef ... int16;
typedef ... uint16;
typedef ... int32;
typedef ... uint32;
typedef ... int64;
typedef ... uint64;

説明

jsint および jsuint は 32 ビットの整数型です。

jsword および jsuword はポインタサイズの整数型です。

intN および uintN はプラットフォームのネイティブの int 型と同じサイズの整数型です。
