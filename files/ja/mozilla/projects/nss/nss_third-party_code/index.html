これは、NSS のリポジトリに含まれるサードパーティ製のコードのリストで、2つのリストに分かれています。NSS ライブラリにコンパイルされるコードと、テストにのみ使用されるコードです。

NSS ライブラリにコンパイルできるコードがすべてそうとは限らないことに注意してください。多くの場合、これはビルド時に様々なトレードオフを考慮して設定することができます。

コンパイル

  • sqlite [/lib/sqlite]
  • BerkleyDB [/lib/dbm]
  • zlib [/lib/zlib]
  • libjar [/lib/jar]
  • Fiat-Crypto, Ring [lib/freebl/ecl]

テストに使用

  • GTest [/gtests]

特定のテストツールでダウンロード

  • TLSFuzzer [/tests/tlsfuzzer]
  • BoGo tests [/tests/bogo]
  • BoringSSL, OpenSSL [/tests/interop]
