概要

このユーティリティクラスは、 C++ のコードからの XPCOM インタフェースの参照の管理をシンプルにします。

#include "nsCOMPtr.h"

備考

詳細については『 nsCOMPtr の使用』 を参照して下さい。

このクラスを使用するには、コンポーネントまたはアプリケーションから XPCOM glue library にリンクする必要があります。クラス自体とそれに使用されている支援機能は凍結されていませんが、容易な再利用のために XPCOM glue library の一部として提供されています。
