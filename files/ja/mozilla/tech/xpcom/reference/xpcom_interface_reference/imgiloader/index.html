以下のコードスニペットを使用して imgLoader のインスタンスを生成することができます。

var imgLoader = Components.classes["@mozilla.org/image/loader;1"]
                          .getService(Components.interfaces.imgILoader);

インタフェースは以下のメソッドがあります:

  • loadImage
  • loadImageWithChanel
  • supportImageWithMimeType
