nsIFeedPerson インタフェースは RSS または Atom フィードの著者か貢献者を記述します。

属性

属性 型 説明
email AString その人の email アドレス。
name AString その人の名前。
uri nsIURI その人に関連づけられた URI。ほとんどの場合、これはその人のホームページです。
