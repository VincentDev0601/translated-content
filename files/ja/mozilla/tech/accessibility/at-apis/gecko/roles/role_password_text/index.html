説明

パスワード、またはテキストコンテンツがユーザーに視覚的に示されていない他の場所に使用するテキストオブジェクトです。

マッピング

  • AT-SPI: ROLE_PASSWORD_TEXT
  • ATK: ATK_ROLE_PASSWORD_TEXT
  • UA: NSAccessibilityTextFieldRole
  • MSAA/IA2: ROLE_SYSTEM_TEXT

使用

  • HTML: <input type="password"/>
