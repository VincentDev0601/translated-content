フレームの役割。タイトルバー、ボーダー、メニューバーなどを持つトップレベルウィンドウ。 これは、多くの場合、アプリケーションのための主要なウィンドウとして使用されます。

マッピング

  • AT-SPI: ROLE_FRAME
  • ATK: ATK_ROLE_FRAME
  • UA: NSAccessibilityUnknownRole (unused on Mac OS X)
  • MSAA : ROLE_SYSTEM_APPLICATION
  • IA2: IA2_ROLE_FRAME

使用

使用されていません。
