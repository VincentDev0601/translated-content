« Gecko Roles ページ

ウィンドウ枠を表します。これは、タイトルバーやクライアント、ウィンドウに含まれる他のオブジェクトなどの子オブジェクトを含みます。

別名

  • AT-SPI: ROLE_WINDOW
  • ATK: ATK_ROLE_WINDOW
  • UA: NSAccessibilityWindowRole
  • MSAA/IA2: ROLE_SYSTEM_WINDOW

使用する要素

ロールは、Microsoft Windows により自動的にサポートされています。
