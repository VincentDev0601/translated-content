« Gecko Roles ページ

セルの行と列を含むテーブルを表します。また、任意で行ヘッダと列ヘッダが含まれます。次のロールも参照してください: ROLE_COLUMNHEADER, ROLE_ROWHEADER, ROLE_COLUMN, ROLE_ROW, ROLE_CELL。

別名

  • AT-SPI: ROLE_TABLE
  • ATK: ATK_ROLE_TABLE
  • UA: NSAccessibilityGroupRole
  • MSAA/IA2: ROLE_SYSTEM_TABLE

使用する要素

  • ARIA: grid
  • HTML: <table>
  • XUL: <listbox/>
