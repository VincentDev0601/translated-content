ツールバーを表します。よく使用される機能への簡単なアクセスを提供するコントロール (プッシュボタンやトグルボタン) のグループです。

子要素

インターフェイス

  • nsIAccessible
  • nsIAccessNode

別名

  • AT-SPI: ROLE_TOOL_BAR
  • ATK: ATK_ROLE_TOOL_BAR
  • UA: NSAccessibilityToolbarRole
  • MSAA/IA2: ROLE_SYSTEM_TOOLBAR

使用する要素

  • ARIA: toolbar
  • XUL:<toolbar/>
