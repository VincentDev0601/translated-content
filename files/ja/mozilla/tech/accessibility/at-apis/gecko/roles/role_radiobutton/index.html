« Gecko Roles ページ

ラジオボタンとも呼ばれるオプションボタンを表します。これは、相互に排他的なオプションのグループの1つです。この属性を持つ単一の親を共有するすべてのオブジェクトは、単一の相互排他的なグループの一部であると仮定されます。

マッピング

  • AT-SPI: ROLE_RADIO_BUTTON
  • ATK: ATK_ROLE_RADIO_BUTTON
  • UA: NSAccessibilityRadioButtonRole
  • MSAA/IA2: ROLE_SYSTEM_RADIOBUTTON

使用

  • ARIA: radio
  • HTML: <input> of type radio.
  • XUL: <radio/>
