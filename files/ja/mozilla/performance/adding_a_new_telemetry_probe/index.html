Firefox では、テレメトリシステムが Firefox のパフォーマンス、ハードウェア、使用状況、カスタマイズのさまざまな測定値を収集し、Mozilla に送信します。 この記事では、ドキュメントへのポインタを提供します。

メモ: Firefoxのすべての新しいデータコレクションには、データコレクションピアからのデータコレクションレビューが必要になりました。データピアの1つに feedback? フラグを設定するだけです。1営業日以内に返信を試みます。

はじめに

新しい Telemetry プローブの追加は、Mozilla Source Tree のドキュメントに記載されています。

その他のテレメトリツールとドキュメントを見つけるには、テレメトリポータルにアクセスしてください。

助けを得る

テレメトリチームが問題の解決を支援します。私たちに接触することができます:

  • IRC の #telemetry
  • Slack の #fx-metrics
  • the fx-data-dev mailing list
  • flags for one of the peers on Bugzilla or direct e-mail
