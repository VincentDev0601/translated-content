多くの人にとって、モバイル端末は Web にアクセスする第一の方法であり、場合によっては唯一の方法になってきています。iOS 版 Firefox (Android 版と同様にコードネームは Fennec です) は、デスクトップ版 Firefox のようにオープンで、ハック可能な、ウェブ標準に準じたブラウザーです。

iOS 版 Firefox は XUL の代わりにネイティブなウィジェットでUIを構築しています。これは大幅に性能、特に起動時間、メモリー消費量を改善させます。Apple の App Store の制限のため、iOS 上では iOS 組み込みの WKWebView も使用しています。これは Gecko でなく WebKit をベースにしています。

iOS 版 Firefox に貢献する

我々は iOS 版 Firefox 作成に力を入れており、また我々はあなたの助けを求めています。iOS 版 Firefox プロジェクトへの貢献に興味があるの主な出発点は GitHub のページ にあります。

iOS 版 Firefox の構築、改善を以下によって手助けすることができます:

  • mobile-firefox-dev メーリングリスト の購読
  • IRC 上のチームに連絡
  • Wednesday development meeting への参加
  • Planet Firefox Mobile、Tumblr や Twitter からプロジェクトの最新情報の取得
  • iOS 版 Firefoxのビルドとハック (Fennec)

iOS 版 Firefox でリンクを開く

iOS ではデフォルトのブラウザーを変更できません。そのため、サードパーティー製のブラウザーを使用するユーザーは、コピー・ペーストを使用する必要があります。代わりに、ユーザーを幸せになるためにネイティブアプリにおいていくつかできる方法があります:

  • ユーザーへの iOS アプリの拡張システムを利用できるようにします。iOS 版 Firefoxはあなたのアプリからオープンすること、ブックマークを作成すること、Firefox のリーディングリストに追加することなどの、いくつかのアプリの機能拡張を提供します。NSHipster は、あなたのアプリに UIActivityViewController を追加する方法の 詳しい概要 を説明しています。
  • 簡単に Firefox がインストールされているかどうかを検出し、表示するためのオプションとして、Open-In-Firefox SDK を使用してください。今後のリリースでは、それが簡単に、ユーザーが Firefox で記事を読んだ後に、簡単にあなたのアプリに戻る方法を提供するつもりです。

アドオン

iOS 版 Firefox は現在の iOS 拡張エコシステムをサポートしていますが、デスクトップ版や Android 版 Firefox のような JavaScript のアドオンを (まだ) サポートしていません。

{{bug(1210119)}} で 軽量テーマ に取り組んでおり、{{bug(1466496)}} で WebExtension ベースのテーマ に取り組んでいます。
