IPDL は "IPC(Inter-process communication : プロセス間通信) Protocol Definition Language " の略称で、Mozilla が定義した C++ でプロセス・スレッド同士でメッセージをやり取りする系統立てた安全な手段です。Firefox 内部のマルチプロセスプラグインやタブ向けの、すべてのメッセージは IPDL 言語によって定義されています。
