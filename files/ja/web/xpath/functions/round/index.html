{{ XsltRef() }}

round 関数は、与えられた数値に最も近い整数を返します。

{{ 英語版章題("Syntax") }}

構文

round(decimal )

{{ 英語版章題("Arguments") }}

引数

decimal
丸められる小数。

{{ 英語版章題("Returns") }}

戻り値

decimalよりも小さいか、大きいか、等しい、最も近い整数。

{{ 英語版章題("Notes") }}

注記

  • -0.5 は負のゼロに丸められます。 0.4 は正のゼロに丸められます。

{{ 英語版章題("Defined") }}

定義

XPath 1.0 4.4

Gecko でのサポート
