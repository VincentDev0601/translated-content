{{ XsltRef() }}

count 関数はノード集合に含まれるノードの数を数え、その整数を返します。

{{ 英語版章題("Syntax") }}

構文

count(node-set )

{{ 英語版章題("Arguments") }}

引数

node-set
カウント対象のノード集合。

{{ 英語版章題("Returns") }}

戻り値

ノード集合に含まれるノードの数を表す整数。

{{ 英語版章題("Defined") }}

定義

XPath 1.0 4.1

{{ 英語版章題("Gecko support") }}

Gecko でのサポート

サポート済み。
