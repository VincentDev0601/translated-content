{{ XsltRef() }}

false 関数は真偽値 false を返します。

{{ 英語版章題("Syntax") }}

構文

false()

{{ 英語版章題("Returns") }}

戻り値

真偽値 false。

{{ 英語版章題("Notes") }}

注記

この関数は比較の一部として役立ちます。

<xsl:if test="boolean((1 &gt; 2) = false())">
 The expression evaluates as true
</xsl:if>

{{ 英語版章題("Defined") }}

定義

XPath 1.0 4.3

Gecko でのサポート
