{{ XsltRef() }}

string-length 関数は、与えられた文字列の文字数に等しい数値を返します。

{{ 英語版章題("Syntax") }}

構文

string-length( [string] )

{{ 英語版章題("Arguments") }}

引数

string(省略可)
評価する文字列。省略すると、コンテキストノードを文字列に変換したものが使用されます。

{{ 英語版章題("Returns") }}

戻り値

文字列の文字数に等しい整数。

{{ 英語版章題("Defined") }}

定義

XPath 1.0 4.2

{{ 英語版章題("Gecko support") }}

Gecko でのサポート

サポート済み。
