{{ XsltRef() }}

normalize-space 関数は文字列から前後の空白文字を取り除き、連続する空白文字を 1 つのスペースに置き換え、その結果として得られる文字列を返します。

{{ 英語版章題("Syntax") }}

構文

normalize-space( [string] )

{{ 英語版章題("Arguments") }}

引数

string (省略可)
正規化する文字列。省略すると、コンテキストノードを文字列に変換したものが使用されます。

{{ 英語版章題("Returns") }}

戻り値

正規化された文字列。

{{ 英語版章題("Defined") }}

定義

XPath 1.0 4.2
