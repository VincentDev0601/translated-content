{{ XsltRef() }}

not 関数は式を真偽値として評価し、その逆の値を返します。

{{ 英語版章題("Syntax") }}

構文

not(expression )

{{ 英語版章題("Arguments") }}

引数

expression
この式は boolean() 関数の引数として渡された場合とまったく同じように評価されます。

{{ 英語版章題("Returns") }}

戻り値

false と評価できる式に対しては true を返し、true と評価できる式に対しては false を返します。

{{ 英語版章題("Notes") }}

注記

  • この関数は boolean() 関数と同様のふるまいをしますが、返される値はその逆です。

{{ 英語版章題("Defined") }}

定義

XPath 1.0 4.3

{{ 英語版章題("Gecko support") }}

Gecko でのサポート

サポート済み。
