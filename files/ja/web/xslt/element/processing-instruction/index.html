<xsl:processing-instruction> 要素は処理命令を出力文書に書き込みます。

構文

<xsl:processing-instruction name=NAME> TEMPLATE </xsl:processing-instruction>

必須属性

name
この処理命令の名前を指定します。

任意属性

なし

タイプ

命令は、テンプレート内に表示されます。

定義

XSLT, section 7.3

Gecko のサポート

サポート済み
