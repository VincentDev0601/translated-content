ピタゴラスの定理を証明します: a 2 + b 2 = c 2

大きな正方形の面積と、内側の正方形の面積 (直角三角形の斜辺の二乗)と 4 つの三角形の面積の合計が等しいことを示すことにより、この定理を代数的に証明できます: ( a + b ) 2 = c 2 + 4 ⋅ ( 1 2 a b ) a 2 + 2 a b + b 2 = c 2 + 2 a b a 2 + b 2 = c 2
