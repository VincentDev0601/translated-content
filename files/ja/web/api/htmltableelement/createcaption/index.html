概要

createCaption は、テーブル内に新たに {{htmlelement("caption")}} 要素を生成します。

構文

HTMLTableElement = table.createCaption()

例

mycap = mytable.createCaption();

注記

既にテーブル内に caption 要素が存在した場合、このメソッドはその caption 要素を返します。
