{{APIRef("Shadow DOM")}}

Slotable mixin は、ノードを {{htmlelement("slot")}} 要素のコンテンツにする機能を定義します。— 次の機能が {{domxref("Element")}} と {{domxref("Text")}} の両方に含まれています。

プロパティ

{{domxref("Slotable.assignedSlot")}} {{readonlyInline}}
ノードが挿入されている {{htmlelement("slot")}} を返します。

メソッド

なし。
