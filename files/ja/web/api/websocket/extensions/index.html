{{APIRef("Web Sockets API")}}

WebSocket.extensions は、サーバーによって選択された拡張機能を返す読み取り専用のプロパティです。現在は、空の文字列か、接続によってネゴシエートされた拡張機能のリストのみを返します。

構文

var extensions = aWebSocket.extensions;
