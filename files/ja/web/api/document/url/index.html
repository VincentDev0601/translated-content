{{domxref("Document")}} インターフェイスの URL プロパティは読取専用で、文書の位置を文字列で返します。

構文

var string = document.URL

例

JavaScript

document.getElementById("url").textContent = document.URL;

HTML

<p id="urlText">
  URL:<br/>
  <span id="url">URL goes here</span>
</p>
