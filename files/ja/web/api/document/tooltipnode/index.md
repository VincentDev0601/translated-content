**`Document.tooltipNode`** プロパティは、現在の {{XULElem("tooltip")}} の対象となるノードを返します。

## 構文

```
document.tooltipNode;
```

## 仕様書

XUL で定められたメソッドであり、標準仕様には含まれていません。{{Source("dom/public/idl/xul/nsIDOMXULDocument.idl#59", "nsIDOMXULDocument.idl")}} で定義されています。
