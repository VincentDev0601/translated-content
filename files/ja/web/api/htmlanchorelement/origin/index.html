{{APIRef("URL API")}}

HTMLHyperlinkElementUtils.origin 読み取り専用プロパティは、表現された URL のオリジンの Unicode シリアル化を含む {{domxref("USVString")}} です。 それは、

  • http または https を使用する URL の場合、スキームの後に '://'、ドメイン、':'、ポートの順になります。 （ポート部分は、明示的に指定されていて、それがデフォルトポートでない場合に存在します。 それぞれのデフォルトポートは、80 と 443 です。）
  • file: スキームを使用する URL の場合、値はブラウザーに依存します。
  • blob: スキームを使用する URL の場合、blob: に続く URL のオリジンです。 例えば、"blob:https://mozilla.org" では "https://mozilla.org" です。

{{AvailableInWorkers}}

構文

string = object.origin;

例

// このページのオリジンを返します
var result = window.location.origin; // 戻り値: 'https://developer.mozilla.org'
