{{APIRef("CSSOM")}}

CSSStyleSheet.deleteRule() メソッドは、スタイル規則を現在のスタイルシートオブジェクトから削除します。

構文

stylesheet.deleteRule(index)

引数

  • index は、規則の位置を表す倍精度整数値。

戻り値

無効。

例

 myStyles.deleteRule(0);
