読み取り専用のプロパティ ValidityState.tooLong は、 {{HTMLElement("input")}} や {{HTMLElement("textarea")}} の値が、ユーザーの編集によって、その要素の maxlength 属性で設定された最大コード単位長を超えているかどうかを示します。
