概要

ソーシャルメディアプロバイダのコンテンツサイドバーが現在表示されているかどうかを示します。

構文

var visible = navigator.MozSocial.isVisible

サイドバーが表示されている場合、visible は true に、非表示の場合 false となります。

仕様書

どの仕様書にも含まれていません。
