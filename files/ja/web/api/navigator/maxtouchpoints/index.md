{{APIRef("HTML DOM")}}

{{domxref("Navigator")}} インターフェイスの **`maxTouchPoints`** プロパティは読み取り専用で、現在の端末で対応している同時タッチコンタクト点の最大数を返します。

## 構文

```
touchPoints = navigator.maxTouchPoints;
```

## 例

```js
if (navigator.maxTouchPoints > 1) {
  // ブラウザーがマルチタッチに対応
}
```
