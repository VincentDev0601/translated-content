{{APIRef("DOM")}}

DOMString は、UTF-16 の文字列です。 JavaScript では同様の文字列を用いるため、 DOMString は直接 {{jsxref("String")}} に対応づけられます。

DOMString を受け付けるメソッドや引数に {{jsxref("null")}} を渡すと、一般的に "null" として文字列化します。
