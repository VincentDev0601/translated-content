{{domxref("PushMessageData")}} インターフェースの text() メソッドは、プッシュメッセージデータをプレーンテキスト文字列として抽出します。

構文

var myText = PushEvent.data.text();

パラメーター

なし。

戻り値

{{domxref("USVString")}}。

例

self.addEventListener('push', function(event) {
  var textObj = event.data.text();

  // テキストを使用して何かを実行する
});
