概要

{{domxref("window")}} の load イベントに対応するイベントハンドラです。

構文

window.onload = funcRef;
  • funcRef : load イベント発生時に呼び出す関数への参照または関数式

例

window.onload = function() {
  doSomething(); // DOM 構築後に実行したい関数の呼び出し
};

<!DOCTYPE html>
<html lang="ja">
<head>
<meta charset="UTF-8" />
<title>onload のテスト</title>

<script>
function load() {
  alert("load イベントが発生しました。");
}

window.onload = load;
</script>

</head>
<body>
<p>文書の読み込みの完了時に load イベントが発生します。</p>
</body>
</html>

注記

load イベントは文書のローディング工程の終了時に発生します。このイベントが発生した時点で、文書中の全てのオブジェクトは DOM 内にあり、全ての画像とサブフレームのロードは完了しています。

DOMContentLoaded や DOMFrameContentLoaded のような Gecko で定められた DOM イベント (※{{domxref("element.addEventListener()")}} でハンドリング可能) もありますが、これらは他のリソースのロード完了を待たずに発生するものです。つまり、 window.onload より先に発生します。
