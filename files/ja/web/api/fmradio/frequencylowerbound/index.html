概要

frequencyLowerBound プロパティの値は、{{domxref("FMRadio.seek","seek")}} メソッドで検索できるラジオ局の周波数の最小値を示します。

構文

var max = instanceOfFMRadio.frequencyLowerBound

値

MHz (MegaHertz) 単位の数値を返します。

仕様書

仕様はありません。
