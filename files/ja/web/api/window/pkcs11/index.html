概要

pcks11 プロトコル に関連するドライバや他のソフトウェアをインストールするために用いられる pkcs11 オブジェクトを返します。

{{ 英語版章題("Syntax") }}

構文

objRef = window.pkcs11

{{ 英語版章題("Example") }}

例

 window.pkcs11.addModule(sMod, secPath, 0, 0);

{{ 英語版章題("Notes") }}

注

pcks11 オブジェクトの扱い方についてのさらなる情報は、nsIDOMPkcs11 を参照してください。
