概要

コンテンツの大きさにウィンドウのサイズを変更します。

DOM コンテンツはこの関数が呼び出されたときに読み込まれるべきです。例えば、DOMContentLoaded イベント後などです。

構文

window.sizeToContent()

例

window.sizeToContent();
