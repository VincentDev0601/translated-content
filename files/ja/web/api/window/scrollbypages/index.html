{{ ApiRef() }} {{ 英語版章題("Summary") }}

概要

指定したページ分文書をスクロールします。 {{ 英語版章題("Syntax") }}

構文

window.scrollByPages(pages)

{{ 英語版章題("Parameters") }}

引数

  • pages は、スクロールするページ数です。
  • pages は、正、あるいは、負の整数です。

{{ 英語版章題("Example") }}

例

// 1 ページ分文書をスクロールダウンします。
window.scrollByPages(1);

// 1 ページ分文書をスクロールアップします。
window.scrollByPages(-1);
