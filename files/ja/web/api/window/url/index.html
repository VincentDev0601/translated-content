Window.URL プロパティは、オブジェクト URL の作成や操作に用いる静的なメソッドを提供します。

{{AvailableInWorkers}}

構文

静的なメソッドの呼び出し:

img.src = URL.{{domxref("URL.createObjectURL", "createObjectURL")}}(blob);

新しいオブジェクトの構築:

var url = new {{domxref("URL.URL", "URL")}}("../cats/", "https://www.example.com/dogs/");
