{{APIRef("DOM")}}

**`Document.onoffline`** イベントハンドラーは、 {{event("offline")}} が {{HtmlElement("body")}} 要素上で発生してバブリングしたとき、 {{domxref("navigator.onLine")}} プロパティが変化して `false` になったときに呼び出されます。
