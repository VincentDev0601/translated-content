{{ ApiRef() }} {{ 英語版章題("Summary") }}

概要

画面の上端からユーザのブラウザの上端までの垂直距離を返します。

{{ 英語版章題("Syntax") }}

構文

lLoc = window.screenY

{{ 英語版章題("Parameters") }}

戻り値

  • lLoc は、画面の上からのピクセル数です。

{{ 英語版章題("Notes") }}

注記

window.screenX も参照してください。
