ウィンドウディレクトリのツールバーオブジェクトを返します。 {{ 英語版章題("Syntax") }}

構文

dirBar = window.directories

{{ 英語版章題("Parameters") }}

引数

dirBar は、barProp 型のオブジェクトです。 {{ 英語版章題("Example") }}

例

<script>
 function dirs() {
  alert(window.directories);
 }
</script>
