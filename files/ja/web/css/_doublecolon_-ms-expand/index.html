::-ms-expand は CSS の疑似要素で、 Microsoft 拡張であり、 {{HTMLElement("select")}} メニューコントロールでドロップダウンメニューを開いたり閉じたりするボタンを表します。通常、これは下を向いた三角形です。

許可されているプロパティ

以下の CSS プロパティのみが、 ::-ms-expand をセレクターに含む規則で使用することができます。他のプロパティは無視されます。
