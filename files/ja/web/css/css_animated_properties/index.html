CSS アニメーションまたは CSS トランジションを使用すると、特定の CSS プロパティをアニメーションさせることができます。アニメーションとは、それらの値を所定の時間をかけて徐々に変化させることができることを意味します。

アニメーション可能なプロパティは次の通りです。
