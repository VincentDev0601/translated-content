{{Non-standard_header}}{{CSSRef}}

概要

非標準の :-moz-list-bullet Mozilla CSS 擬似クラスは、リスト要素のマーカーを編集するために使用されます。

構文

CSS

.list li::-moz-list-bullet {
  font-size: 36px;
}

HTML

<ul>
  <li>Number 1</li>
  <li>Number 2</li>
  <li>Number 3</li>
</ul>
<ul class="list">
  <li>Number 1</li>
  <li>Number 2</li>
  <li>Number 3</li>
</ul>

結果
