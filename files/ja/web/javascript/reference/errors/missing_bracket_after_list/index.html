メッセージ

SyntaxError: missing ] after element list

エラータイプ

{{jsxref("SyntaxError")}}。

何がうまくいかなかったのか？

どこかで配列初期化構文のエラーがあります。閉じ括弧（"]"）かカンマ（","）が不足している可能性があります。

例

不完全な配列初期化

var list = [1, 2,

var instruments = [
  "Ukulele",
  "Guitar",
  "Piano"
};

var data = [{foo: "bar"} {bar: "foo"}];

次のように修正できます：

var list = [1, 2];

var instruments = [
 "Ukulele",
 "Guitar",
 "Piano"
];

var data = [{foo: "bar"}, {bar: "foo"}];
