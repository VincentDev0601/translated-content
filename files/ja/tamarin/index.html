Tamarin は C++ で書かれた JavaScript エンジンです。現在は Adobe ActionScript™ 3 (ECMAScript 第 3 版のスーパーセット) を実装しており Adobe® Flash® Player 9 に搭載されています。Tamarin を Firefox で利用するために SpiderMonkey と統合する計画がありましたが、2008 年にその計画は中止されました。詳細は ActionMonkey プロジェクトをご覧ください。

開発ロードマップに関する詳細は Tamarin プロジェクトページ をご覧ください。

Tamarin は MPL/GPL/LGPL トリプルライセンスの下で利用できます。
