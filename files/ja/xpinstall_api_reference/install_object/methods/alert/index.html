概要

引数に指定したメッセージを持つモーダルダイアログボックスを表示します。

Method of

Install オブジェクトのメソッド

構文

void alert ( String message );

引数

message 引数はダイアログボックス内の文字列として表示されます。メッセージが文字列でない場合は、変換されて表示されます。

戻り値

無し
