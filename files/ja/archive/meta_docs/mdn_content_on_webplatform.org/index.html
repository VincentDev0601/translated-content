Web Platform Docs (WPD) は Web の標準に関するドキュメンテーションと教育のためのサイトです。 W3C によって運営されており、Mozilla や他の組織が支援しています。MDN から WebPlatform.org への MDN の Creative Commons Share-Alike 2.5 (CC-BY-SA) ライセンスのもとでの貢献は誰でも歓迎します。MDN で作られたコンテンツは MDN に所有権があるため、もととなった記事へのリンクがなければなりません。Web Platform Docs サイトには、これらの作業をシステマティックに解決できるようなサポートがあります。

WPD のアカウントを作り、新しいページを作成したら、"new page" のオプションとして、コンテンツが MDN に由来するものであると指定できます。WPD の記事の一部の sections のみを MDN 由来のものとマークすることもできます。その場合は、セクションは CC-BY content に統合されます。(CC-BY が WPD のデフォルトのライセンスだからです。) WebPlatform.org の External Attribution ページに作業方法について書かれています。
