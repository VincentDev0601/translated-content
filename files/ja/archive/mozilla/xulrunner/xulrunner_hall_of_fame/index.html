XULRunner ベースのアプリケーション

Abstract
画期的な自動化をもたらす会計パッケージ。
ChatZilla
スタンドアロン版の ChatZilla IRC クライアント。
Clines
Color Lines(ゲーム) のクローン。
CocoaPSYC.app
Mac OS X 向けの PSYC IM/チャットクライアント (Firefox の拡張機能としても利用可能)。
Convertigo Enterprise Mashup Server
エンタープライズ・マッシュアップを構築するための、Web 統合・クリップツール。
DAIM (仏) / DAIM (英))
プロフェッショナルな imagelib の頂点に立つ画像解析ビルドのための極上のツール。
eMusic Remote
eMusic Remote は eMusic からスナップをダウンロードします。
Echofon for Windows
Twitter クライアント。Firefox 用拡張機能 も公開されています。
Evergreen - Staff Client
Evergreen はオープンソースの図書館の蔵書検索システムです。
FindThatFont!
フォントを探すのに便利な Windows および Linux 向けのフォント管理ツールです。
Flickr Uploadr 3.0
あなたの写真を Flickr にアップロードします。
GencatRss
RSS フィードマネージャ。
Google AdWords Editor
AdWords Editor は、Google 製のフリーでダウンロード可能な、あなたのコンピュータ向けのアカウント管理アプリケーションです。
IDA
簡単に Web ベースの学習アプリケーションを作成できる e ラーニング支援システム。
Instantbird
複数のプロトコルに対応した IM クライアント。
Joost (以前のコードネームは The Venice Project)
"TV の大好きなところが、インターネットの相互力によってすべて溶け合ったもの"
Kirix Strata
Web からデータにアクセスして操作する、新しい特別なブラウザ。
Kneemail
オープンソースの、祈りと賛美、一般的な目的のための日記アプリケーション。
Mango
Jabber クライアント。
Miro Media Player (Democracy Player として知られていたもの)
"フリーでオープンソースの TV プラットフォーム"
Plain Old Webserver
サーバサイド JavaScript で動作するフル機能の Web サーバ。
Raptr
最新のパッチと更新を取り入れて、ゲームを最新の状態に保つとともに、ゲーム中心型のソーシャルネットワークである Raptr のゲームプレイをトラックするデスクトップクライアント。
Songbird
"Songbird™ は、デジタルジュークボックスと Web ブラウザがマッシュアップされたデスクトップ Web プレイヤーです。"
Spicebird
E メール、カレンダー、インスタントメッセージのコラボレーションスイート。
TeleKast
オープンソースのテレプロンプターおよびスクリプトエディタです。
TomTom HOME 2
TomTom GPS 機器を管理する PC アプリケーション。
WengoPhone
SIP/XMPP ベースの IM/VoIP クライアント。
Lotus Notes / Sametime
IBM Lotus Notes および Sametime の最新版は XUL アプリケーションを組み込むことができます。
Embed XULRunner in Java
ブラウザ中立 Java API の提供を目的としています。
Komodo Edit
複数の言語コードに対応した開発ツール。

ユーティリティ / 試作版

AJAX Toolkit Framework (ATF)
Eclipse Web Tools Platform (WTP) の一部。
Benjamin's XULRunner examples
"mybrowser is a very simple example browser", xulmine
Exch
http://finance.yahoo.com/currency の更新データを利用した為替ツール。Firefox の拡張機能かスタンドアローンの XUL アプリケーションを使用できます。
OpenDocument Viewer
クロスプラットフォームの OpenDocument ファイルのためのビューア。
TaskPool
TaskPool は時間を節約する小さなアプリケーションです。タスクをプールに追加し、それらのオンとオフを簡単に切り替えます。
SVGclock
SVG ベースの時計。
Crowbar
サーバ側のヘッドレス mozilla ベースブラウザーの使用に基づくWebスクレイピング環境。
30b
30boxes.com 向けの XULRunner ベースのラッパー。
XUL Explorer
"軽量の XUL 開発環境... XUL の実験をする簡単な方法"
Foxkeh Clock (日本語)
フォクすけ のキャラクタをベースにした時計。
Prism (Webrunner として知られていたもの)
簡単で小さな Web ブラウザシェル、または Gmail を別のブラウザプロセスで実行するような Web アプリケーションに便利な "特定のサイトブラウザ"。
