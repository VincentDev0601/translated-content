XULRunner 1.8.0.1 は、XULRunner 初の「開発者向け安定版プレビュー」リリースです。ネイティブアプリケーションおよび Java アプリケーション内で Web ページを描画する際に使用できる、スタンドアローン XUL アプリケーションや埋め込み API の起動基盤を提供します。

セキュリティ/安定性を向上した XULRunner 1.8.0.4 が現在入手可能です; すべてのユーザはこの新しいバージョンをインストールすべきです。

システム要件

インストール前に、お使いのコンピュータが システム要件 を満たしているか、必ず確認してください。XULRunner のシステム要件は、Firefox のシステム要件と同じです。

XULRunner のインストール

mozilla は、XULRunner を Windows、Mac OS X、そして Linux を対象に、英語で提供しています。

  • Download XULRunner for Windows
  • Download XULRunner for Mac OS X
  • Download XULRunner for Linux

Windows

  • zip ファイルを、解凍ツール（7zip を推奨します）を使って新しいディレクトリに展開します。次のディレクトリをおすすめします：<tt>C:¥Program Files¥Mozilla XULRunner¥1.8.0.1</tt>
  • XULRunner をシステムに登録するには、コマンドプロンプトを開いて、<tt>xulrunner.exe --register-global</tt> （全ユーザを対象に登録）または <tt>xulrunner.exe --register-user</tt> （単一ユーザのみに登録）を実行します。

Mac OS X

  • .pkg ファイルをインンストーラ内で展開して、指示に従ってください。

Linux

  • <tt>tar -xzf xulrunner-1.8.0.1.en-US.linux-i686.tar.gz</tt> として、tarball を新しいディレクトに展開します。次のディレクトリをおすすめします：<tt>/opt/xulrunner/1.8.0.1</tt>
  • XULRunner をシステムに登録するには、<tt>xulrunner --register-global</tt>（全ユーザを対象に登録）または <tt>xulrunner --register-user</tt>（単一ユーザのみに登録）を実行します。

XULRunner のアンインストール

Windows/Linux

  • コマンドプロンプトから <tt>xulrunner --unregister-global</tt> または <tt>xulrunner --unregister-user</tt> を実行して、インストール時に登録したのと同じように、XULRunner の登録を解除します。
  • XULRunner ディレクトリを削除します。

Mac OS X

Mac OS X に XULRunner が複数インストールされている場合は、あるバージョンのみをアンインストールすることはできません。

  • インストールされている すべて の XULRunner を削除するには、<tt>/Library/Frameworks/XUL.framework</tt> ディレクトリを削除します。

XUL アプリケーションのインストール

XUL アプリケーションは、さまざまな場所から入手でき、通常は <tt>.xulapp</tt> や <tt>.xpi</tt> といった拡張子が付いた ZIP アーカイブとしてパッケージされています。Mozilla Developer Center から サンプルアプリケーション が提供されています。

  • デスクトップなどお好みの場所にアプリケーションアーカイブを保存します。

Windows

「スタート」 -> 「ファイル名を指定して実行」またはコマンドプロンプトから次のコマンドを実行します： 

"C:¥Program Files¥Mozilla XULRunner¥1.8.0.1¥xulrunner¥xulrunner.exe" --install-app "C:¥Documents and Settings¥User¥Desktop¥myapplication.xpi"

アプリケーションは <tt>C:¥Program Files¥VendorName¥ApplicationName</tt> にインストールされます。

Mac OS X

次のコマンドをコマンドプロンプトで実行します： 

/Library/Frameworks/XUL.framework/xulrunner-bin --install-app ~/Desktop/myapplication.xpi

アプリケーションは <tt>/Applications/Vendor/AppicationName</tt> にインストールされます。

Linux

次のコマンドをコマンドプロンプトで実行します： 

/opt/xulrunner/1.8.0.1/xulrunner/xulrunner --install-app ~/Desktop/myapplication.xpi

アプリケーションは <tt>/usr/lib/vendor/applicationname</tt> にインストールされます。

既知の問題

  • ファイアウォールソフトウェアが、すべての XUL アプリケーションを同一アプリケーションと認識する可能性があります。
  • Microsoft Windows XP では、すべての XUL アプリケーションがタスクバーにまとめられてしまう可能性があります。
