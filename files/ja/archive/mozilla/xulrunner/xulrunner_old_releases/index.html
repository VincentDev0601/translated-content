Nightly builds of XULRunner 1.9.0.x

コミュニティから寄贈された XULRunner 1.8.1.3
これらのビルドは Eclipse ATF コミュニティによって寄贈され、組み込み用途でビルドされたものです。Windows や Linux ビルドは通常の XULRunner アプリケーションを実行しますが、Mac ビルドは Cocoa ウィジェットを使用するため XULRunner アプリケーションを実行できません。
XULRunner 1.8.0.4
これは 1.8.0 branch からの開発者向けプレビューリリースの最後の公式安定版です。これには既知のセキュリティホールがあるため、パブリックな Web コンテンツを扱うアプリケーションで使用してはいけません。
