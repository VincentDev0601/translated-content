--- title: toolkit.defaultChromeFeatures slug: Archive/Mozilla/XULRunner/toolkit.defaultChromeFeatures tags: - Preferences translation_of: Archive/Mozilla/XULRunner/toolkit.defaultChromeFeatures ---

Type: String
Specified by: default prefs of XULRunner applications
Default: "chrome,dialog=no,all"
Example: pref("toolkit.defaultChromeFeatures", "chrome,resizable=no,dialog=no");

toolkit.defaultChromeFeatures preference は、単一の XULRunner ベースのアプリケーションに対して、 window.open() が渡されたときにメインアプリケーションウィンドウがどのような特性で開かれるのかを指定することができます。

参照: XULRunner:Specifying Startup Chrome Window, DOM:window.open, toolkit.defaultChromeURI.
