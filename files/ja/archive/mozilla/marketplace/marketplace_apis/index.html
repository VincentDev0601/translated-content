Firefox Marketplace が提供する API の主なレファレンスへのリンク集です。アプリの登録や支払いを受けるアカウントの変更といったことを、API から行えます。

Marketplace 向けユーティリティライブラリ
アプリ内課金や、レシートデータを利用した支払いの検証を簡単に行える JavaScript ライブラリの紹介。
登録 API
この API を利用することで、アプリの検証、更新やアプリのアップデートの有無の確認が行えます。
課金 API
アプリ内課金の情報や、様々な国におけるアプリの価格帯に関する情報などを取得できる API です。
その他の API
全 Firefox Marketplace API のドキュメント。
