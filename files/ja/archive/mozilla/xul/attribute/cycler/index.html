cycler
型: 論理型
true の場合、列は cycler column になります。例えば列内のセルをクリックすると、その on と off の状態が交互に切り替わります。これは例えば、メールウィンドウ内でメッセージが既読かどうかをセル内の小さなマークで示すために使用されます。cycler 属性が設定されていないときは、セルは通常のテキストセルになります。
