disableautoselect
型: 論理型
この属性が true または省略された場合、menu 上の選択された項目は、ユーザが textbox に入力したものと一致させるために更新されます。text が list 内のどの項目とも一致しない場合、menu の選択がクリアされます。この属性が false の場合、選択された項目は更新されません。この属性は editable な menulist にのみ適用されます。
