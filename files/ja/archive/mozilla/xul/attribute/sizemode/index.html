sizemode
型: 下記の値のいずれか一つ
window の状態。次の値が使用されます:
  • maximized: window は最大化されており、スクリーン全体を占有しています。
  • minimized: window は最小化されているか隠れています。
  • normal: window は定められたサイズで通常の状態です。
