noautohide
型: 論理型
この属性が false に設定されるか省略された場合、数秒経つと tooltip は自動的に消えます。この属性が true に設定された場合、このような動作は起こらず、ユーザがマウスを他の要素に移動した時、ツールチップが隠れます。
