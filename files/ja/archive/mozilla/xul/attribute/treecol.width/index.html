width
型: 文字列型 (整数型に相当する)
優先される tree コラムの幅。すべての値はピクセル単位なので、値に単位を含めてはいけません。列に利用可能なスペースよりも広い幅を指定すると、Gecko が tree を水平スクロールできるようにします。
