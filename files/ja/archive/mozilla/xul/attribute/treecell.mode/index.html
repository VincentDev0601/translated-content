mode
型: 下記の値のいずれか一つ
進捗メータの列で使用する進捗メータの形式を決定します。
  • none: 進捗メータは表示されません。代わりにセルのテキストが表示されます。
  • normal: セルは進捗バーの最大量を決定するために、その value 属性を使用します。
  • undetermined: progressmeter は未確定です。
