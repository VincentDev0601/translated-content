uri
型: 文字列型
template が生成するコンテンツのために、この属性は、コンテンツが生成を開始する要素上に置かれます。そのため、このコンテンツは template の子孫要素上に置かれます。値は rdf:* に設定されます。
この属性で要素内に現れる要素は、RDF データソース内の各ノードが繰り返されます。外部の要素は一度だけ現れます。
