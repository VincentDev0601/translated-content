tooltip
型: tooltip 要素の id
要素の上にマウスを合わせて少し経った時にツールチップウインドウとして使用される popup要素の id 属性の値を指定します。tooltip はマウスが動かされた時に自動的に消えます。この属性に '_child' を指定した場合、その要素の最初の tooltip 子要素が使用されます。
