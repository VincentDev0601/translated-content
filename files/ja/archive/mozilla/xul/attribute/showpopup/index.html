showpopup
型: 論理型
true の場合 popup が表示されます。false の場合 popup は表示されませんが、自動補完の候補が利用可能です。この属性を false に設定して autoFill 属性を true に設定すると、Communicator 4.x スタイルの自動補完 textbox をエミュレートすることができます。デフォルト値は true です。
