preference-editable
Mozilla 1.8
型: 論理型
true の場合、要素は prefwindow 内の設定を変更するものとして使用されます。preference 属性が preference 要素への接続に使用されます。これは XBL で実装されたカスタム要素に役立ちます。値が変更されると要素が change イベントや select イベントを引き起こすので、それに応じて preference も更新されます。
