src
型: URL
要素内に表示するコンテンツの URL。

例

<iframe id="content-body" src="http://www.mozilla.org/"/>
<browser src="http://www.mozilla.org" flex="1"/>
<image src='Firefoxlogo.png' width='135' height='130'/>
