insertbefore
型: 要素の id
要素が overlay 内にある時、その要素はベースウィンドウ内で insertbefore 属性で指定した id を持つ要素の前に現れます。この値はカンマで区切られた id のリストです。この id は window 内で最初に見つかったものが使用されます。
