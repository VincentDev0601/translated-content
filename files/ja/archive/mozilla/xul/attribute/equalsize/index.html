equalsize
型: 下記の値のいずれか一つ
この属性は、要素の子を等しいサイズにするために使用します。
  • always: 水平指向の要素では、すべての子要素の幅が等しくなります。垂直指向の要素では、すべての子要素の高さが等しくなります。
  • never: すべての子要素が、コンテンツに必要なサイズ、または width 属性と height 属性か、CSS の width プロパティと height プロパティで指定されたサイズで表示されます。
