targets
型: カンマで区切られた要素の id のリスト
command update が更新する要素の id の、カンマで区切られたリスト。この属性が指定されていないか '*' を設定した場合、すべての要素で効果があります。command update は指定の要素の一つにイベントが発生した時のみ起こります。
