handleCtrlTab
型: 論理型
true に設定するか省略した場合、Ctrl と Tab キーが同時に押されると tabbox が次のタブに切り替えられます。Shift キーも同時に押すと、前のタブが表示されます。この属性を false に設定した場合、これらのキーはタブ操作を行いません。
