fixed
型: 論理型
true の場合、tree 内の列のサイズをユーザが調節できなくなります。固定幅で列のサイズを維持する間、splitter がこれらの両側の列をサイズ変更します。false または指定されていない場合、ユーザはコラムヘッダをマウスでドラッグして列のサイズを調節することができます。
