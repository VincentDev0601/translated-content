value
型: 整数型
進捗を示す 0 から 100 までの範囲の整数。value を "0" に設定すると空のバーが表示され、"100" に設定すると完全な長さのバーが表示されます。また "25" に設定すると 1/4 の長さのバーになります。数値の後の文字は無視されます。
