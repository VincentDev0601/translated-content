autofill New in Thunderbird 3 要 SeaMonkey 2.0
型: 論理型
true に設定した場合、ユーザが入力したものに最もマッチする値が textbox に入力されます。デフォルト値の false の場合、ユーザが項目を選択するまで値は入力されません。
