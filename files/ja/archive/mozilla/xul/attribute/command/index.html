command
型: 要素の id
要素に監視された command 要素の id をセットします。

例

<command id="cmd_openhelp" oncommand="alert('Help');"/>
<button label="Help" command="cmd_openhelp"/>
<button label="More Help" command="cmd_openhelp"/>

参照
