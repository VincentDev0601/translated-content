value
型: 文字列型
デフォルト値が textbox に入力されます。この属性はデフォルト値のみが保持され、ユーザがテキストを入力しても変更されることはありません。更新された値を取得するには value プロパティを使用してください。数値ボックスのデフォルト値は、0 または min プロパティが返す最小値のどちらか大きいほうです。
