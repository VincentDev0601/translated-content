editable
型: 論理型
column 内のセルの内容を変更できるようにします。特に type="checkbox" の場合に役立ちます。ユーザが編集可能なセルをダブルクリックすると、セルの内容を編集するためのテキストフィールドが現れます。
これを適切に動作させるには tree も editable でなければなりません。編集できなくするには column 内の個々の treecell 要素を editable="false" にします。
