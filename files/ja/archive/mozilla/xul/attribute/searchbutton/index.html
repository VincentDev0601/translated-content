searchbutton
型: 論理型
true の場合、ユーザが search ボタンを押すか Enter キーを押した時のみ、search フィールドのコマンドイベントが発生します。false の場合、ユーザが値を変更するだけでコマンドイベントが発生します。この属性は search 型のテキストボックスにのみ適用されます。
