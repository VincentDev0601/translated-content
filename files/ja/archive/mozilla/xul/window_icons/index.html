注意: この機能はFirefox 67で削除されました。詳細はbug 1531836を参照して下さい。

Firefox 1.5、Thunderbird 1.5、および XULRunner 1.8から、XULウインドウのアイコンを<tt>mywindow.ico</tt> (Windowsの場合) および <tt>mywindow.xpm</tt> (Linuxの場合)として指定することができるようになりました。mywindowはbundleの <tt>chrome/icons/default</tt> サブフォルダ内でアイコンを付けたい<window>のidです。

これらのアイコンは、<tt>add_dir/chrome/icons/default</tt>にあるグローバルアイコンファイルを上書きします。

Older versions

To support older applications, such as Firefox 1.0, you need to copy the icons to <tt>app_dir/chrome/icons/default</tt> manually on the first start (example code).
