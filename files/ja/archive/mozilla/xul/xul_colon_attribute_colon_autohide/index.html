autohide
型: boolean
true が設定された場合、ユーザが Alt キーを押さない限り、ツールバーは不可視になります。デフォルト値の false が設定された場合、ツールバーは表示されます。これは Windows でのみ効果があり、type="menubar" と  menubar 要素と組み合わされる必要があります。
