« XUL リファレンス
next
型: 文字列型の wizardpage の id
このページの後にくる次ページの pageid を設定します。設定すると、Next ボタンが押された時、この pageID を持ったページが表示されます。これは連続した手順を持たない wizard の作成に使用されます。一つのページに next 属性があるときは、最後のページを除き、すべてのページがこの属性を持つべきです。
