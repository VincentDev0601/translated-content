canRewind
型: 論理型
このプロパティは、ユーザが Back ボタンを押して前のページへ戻れるとき、true に設定されます。このプロパティが false の場合、ユーザは前のページへ移動することができません。ユーザが前のページへ戻れることを示すには、このプロパティを true に設定します。前のページへの移動を拒否するには false に設定します。これは Back ボタンを有効または無効にします。
