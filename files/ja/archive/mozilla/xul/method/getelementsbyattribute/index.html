getElementsByAttribute( attrib, value )
戻り値の型: DOM NodeList
最初の引数に与えた属性に 2 番目の引数に与えた値が設定された、すべての子要素の配列を返します。2 番目の引数が '*' の場合は、属性に任意の値が設定されたものとします。
このメソッドは XUL 要素上でのみ使用可能です。 DOM の一部ではないので注意してください。
