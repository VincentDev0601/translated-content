appendGroup( group )
戻り値の型: 戻り値無し
※ Firefox では使用不可
いくつかの新しいタブを既存のタブの次に追加します。引数は、タブに読み込む各ドキュメントオブジェクトの配列です。オブジェクトはスクリプトで定義され、読み込むページの URL の URI プロパティを含みます。 referrerURI プロパティは、リファラページを設定するために任意で使用されます。
