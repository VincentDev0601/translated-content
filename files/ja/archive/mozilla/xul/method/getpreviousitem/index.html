getPreviousItem( startItem, delta )
戻り値の型: 要素
このメソッドはstartItem で指定した項目から与えられた間隔 (delta) だけ前の項目を返します。項目が存在しない場合は null を返します。次の例は someItem より 5 行前の項目を返します: getPreviousItem ( someItem, 5 );
