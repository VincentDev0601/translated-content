loadTabs( uris, loadInBackground, replace )
戻り値の型: 戻り値無し
uris 配列で指定された URI のセットをタブに読み込みます。loadInBackground が true の場合、これらのタブは背面に読み込まれます。replace が true の場合、タブを追加する代わりに、現在表示されているタブが指定した URI に置き換えられます。
