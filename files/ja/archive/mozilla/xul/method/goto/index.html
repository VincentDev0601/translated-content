goTo( pageID )
戻り値の型: 戻り値無し
このメソッドは、現在表示されているページを pageID 引数で指定されたページに変更します。ページは canAdvance や canRewind の値にかまわず変更されます。この関数を使用した場合、onwizardback および onwizardnext のコードは呼び出されません。
