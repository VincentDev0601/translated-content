selectTabAtIndex( index, event )
戻り値の型: 戻り値無し
引数 index で指定した位置のタブを選択します。index には負の値も指定可能です。引数 event が指定されている場合は、デフォルトのイベント処理は行われなくなり、イベントの伝播も発生しません。
