getFormattedString( key, strArray )
戻り値の型: 文字列型
string bundle から、指定 key 名の、フォーマット済みの文字列を返します。%S (大文字) が現れる場所は、与えられた配列内の連続する各要素に置き換えられます。他の書式コードを使用することもできます。
