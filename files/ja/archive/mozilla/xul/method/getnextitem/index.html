getNextItem( startItem, delta )
戻り値の型: 要素
このメソッドは、startItem で指定した項目から与えられた間隔 (delta) だけ後の項目を返します。項目が存在しない場合は null を返します。
次の例は someItem より 2 行後の項目を返します: getNextItem ( someItem, 2 );
