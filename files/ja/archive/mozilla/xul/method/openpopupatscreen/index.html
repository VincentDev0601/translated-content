openPopupAtScreen( x, y, isContextMenu )
戻り値の型: 戻り値無し

x と y で指定された画面上の特定の位置にポップアップを開きます。ポップアップが画面からはみ出るように指定された場合には位置が調整されます。座標 x と y は CSS の pixel 単位で計測されます。
