appendCustomToolbar( name, currentset )
Firefox のみ
戻り値の型: element
toolbox に与えられた name の カスタム toolbar を追加します。デフォルトでは、二番目の引数に、追加する toolbar 項目のカンマで区切られた id のリストが提供されます。このメソッドは作成されたツールバーの DOM 要素を返します。
