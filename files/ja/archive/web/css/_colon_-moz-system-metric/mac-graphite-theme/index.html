:-moz-system-metric(mac-graphite-theme) は、 Mac OS X のシステム設定で「外見」設定パネルの「グラファイト」表示をユーザーが選択している場合に要素に一致します。

このセレクターは、主にテーマ開発者が利用することを想定しています。

メモ: Firefox 58 以降、このメディア特性はウェブコンテンツでは利用できなくなりました。内部 (例えば to XUL / chrome code) でのみ利用できます。詳しくは {{bug(1396066)}} をご覧ください。
