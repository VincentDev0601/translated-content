Mozilla Toolkit は、XUL アプリケーションに高度なサービスを提供するために Gecko 上に構築されたプログラミングインタフェース(API) のセットです。これらのサービスは以下のものを含みます:

  • プロファイルの管理
  • Chrome の登録
  • ブラウズ履歴
  • 拡張機能とテーマの管理
  • アプリケーション更新サービス(AUS)
  • セーフモード
