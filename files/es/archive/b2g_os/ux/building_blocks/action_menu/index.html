Un menú de acciones, como indica su nombre, muestra un listado de acciones, relativas al contenido de la aplicación, a traves del cual el usuario puede elegir qué hacer. Visita la Guía de código para saber cómo implementar un menú de acciones en tu aplicación.

Caracteristicas

  • Abiertos desde botones dentro del contenido de la aplicacion, estos botones aparecen a menudo dentro de barras de herramientas (por ejemplo, el boton de "Compartir" de las aplicaciones de navegador).
  • Los menus de acciones contienen uno o mas elementos.
  • Estos menus se expanden en altura para acomodarese a sus elementos ,hasta el máximo de la altura de la pantalla. Una vez que se alcanza el máximo, el contenido se muestra con scroll vertical. Generalmente, una buena práctica es intentar no incluro más de cinco elementos mas el titulo del menú.
  • La cadena del tíutulo es opcional.
  • El menu se cierra por una de dos:
    • Seleccionar una de las acciones.
    • Pulsar el boton de "Cancelar".

Visuales

Estado de reposo

El boton de acciones abierto, esperando a que el usuario presione una seleccion.

Estado pulsado

Aspecto del menu mientras el dedo del usuario presiona la "Opcion 03".

Variaciones

Existen dos variaciones basicas en el menu de acciones: que el menu tenga o no una cadena de titulo en la parte superior.

El diagrama muestra como trabaja el flujo de interaccion del ususario con el menu de acciones.
