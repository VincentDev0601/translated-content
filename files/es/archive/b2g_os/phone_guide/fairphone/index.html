Advertencia: Este dispositivo todavía no ejecuta Firefox OS, pero este es el objetivo final. Esta página pretende ser un punto central de información para la comunidad de la coordinación del trabajo de portabilidad.

El Fairphone es un smartphone de economía justa; centrado en el abastecimiento, la producción, la distribución, la auto-reparación y el reciclaje de productos electrónicos. No es completamente hardware software libre, pero es compatible con la idea de devolver al menos algún tipo de control de hardware para los usuarios.

Información de publicación del dispositivo

Fairphone ha lanzado dos dispositivos hasta ahora - la FP1 v1 Fairphone (primera edición) fue lanzado el 30 de diciembre del 2013 y el FP1U (segundo lote) fue lanzado el 22 de julio de 2014. Las ventas han llegado a su fin para el dispositivo v1. El v2 Fairphone fue lanzado el 16 de julio 2015 y se puede pedir a Fairphone.

Portabilidad de Firefox OS al dispositivo Fairphone

Esto aún no ha sucedido, pero se está trabajando.

Problemas reales

Fairphone v1 no pudo actualizar su sistema operativo Android personalizado porque el fabricante del chipset no dio a conocer un código abierto de banda actualizada o compilación.

En el Mobile World Congress 2015, Fairphone hizo un llamado a los posibles sistemas operativos alternativos disponibles para el próximo Fairphone v2. Estas conversaciones que se llevaron a cabo dieron por inicio la portabiliad de Firefox OS al dispositivo Fairphone. Una portabilidad de Firefox OS puede ser una manera de salvar al Fairphone v1, así como evitar estas cuestiones para las futuras generaciones de teléfonos. La colaboración de Mozilla con los tres principales proveedores de chipset también ayudaría.

Fairphone v2 es orientado a código abierto con contribuciones de la comunidad a código fuente y a compilación de Fairphone OS.

Especificaciones del dispositivo

Fairphone v1

Usted puede encontrar una lista detallada de las especificaciones en la página de las especificaciones de Fairphone.

Red

  • 2G/GSM: 850/900/1800/1900MHz
  • 3G/WCDMA: 900/2100MHz
  • Wifi: 2.4GHz 802.11b/g/n
  • Bluetooth: v2.1 + EDR / v3.0 + HS
  • SAR calificación: Cabeza: 0.329 W/kg; Cuerpo: 0.693 W/kg

Hardware

  • Mediatek MT6589M (primera edición) / MT6589 @ 1.2 Ghz (segundo lote) Chipset: Quad-core CPU con reproducción de vídeo Full HD
  • Cámara frontal: 1.3 MP para fotos y video llamadas
  • Cámara posterior: 8MP AF (estabilización + sensor de imagen) para fotos y video llamadas
  • Pantalla: 4.3 pulgadas qHD IPS (960x540 pixels) (256 ppi); Pantalla tactil capacitiva
  • 16 GB almacenamiento interno + 1GB of RAM
  • MicroSD: Añade hasta 64 GB de memoria adicional
  • Auriculares jack: A partir de una variedad de formatos
  • A-GPS, además de un E-brújula y giroscopio
  • Double SIM: Un SIM: 3G; una tarjeta SIM: 2G. Formato estándar mini tarjeta SIM
  • Puertos de datos y carga: Estándar micro USB 2.0, Tipo B; USB 2.0
  • Batería: 2000mAh (batería reemplazable)

Software

  • Basado en el sistema operativo Android (4.2 Jelly Bean): Con una interfaz especial Fairphone
  • Acceso Root: acceso de superusuario Inmediata

Fairphone v2

Usted puede encontrar una lista detallada de las especificaciones en la página de las especificaciones de Fairphone.

Red

  • 2G/GSM: 850/900/1800/1900MHz
  • 3G/WCDMA: 900/2100MHz
  • 4G LTE: 900/1900/2100MHz Cat. 4 150 Mbps descargar — Cat. 4 50 Mbps subir
  • Wifi: 2.4GHz 802.11b/g/n/ac hasta 433 Mbps
  • Bluetooth: v4.0 LE
  • SAR calificación: Cabeza: 0.288 W/kg; Cuerpo: 0.426 W/kg

Hardware

  • Qualcomm MSM8974AB @ 2.26 Ghz Chipset: Quad-core CPU Krait 400 con Qualcomm Adreno 330 GPU y reproducción de vídeo Full HD
  • Cámara frontal: 2 MP Omnivision OV2685 para fotos y video llamadas
  • Cámara posterior: 8MP AF Omnivision OV8865 (estabilización + sensor de imagen) para fotos y video llamadas
  • Pantalla: 5 pulgadas Full HD (Gorilla Glass 3 — 0.7mm thick) LCD TFT/IPS (446 ppi); Pantalla tactil capacitiva
  • 32 GB almacenamiento interno eMMC5 + 2GB of RAM LPDDR3
  • MicroSD: SDHC, SDXC, UHS hasta 64 GB de memoria adicional
  • Auriculares jack: A partir de una variedad de formatos; estándar CTIA
  • A-GPS, además de un E-brújula y giroscopio
  • Double SIM: Double-SIM Micro-SIM (3FF), Double-Standby (DSDS)
  • Puertos de datos y carga: Estándar Micro USB 2.0 con soporte OTG
  • Batería: 2420mAh (batería reemplazable)

Software

  • Basado en el sistema operativo Android (5.1 Lollipop): Con una interfaz especial Fairphone
  • Acceso Root: Sin acceso de superusuario
