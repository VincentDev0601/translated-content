Actualizaciones en tu Flame: Le animamos a unirse a la siguiente lista de correo para recibir actualizaciones periódicas sobre el software builds y otras noticias relacionadas a su dispositivo Flame: https://mail.mozilla.org/listinfo/flamenews

Flame es el dispositivo de referencia oficial para el desarrollo, prueba y depuración de Firefox OS y aplicaciones web abiertas. Esta guía proporciona información básica acerca de sus especificaciones de hardware, así como documentación práctica específica experimentando con, desarrollando para, y prueba de software compatible con este teléfono.

El hardware de Flame ofrece un conjunto representativo de especificaciones — incluyendo pantalla FWVGA y procesador de doble núcleo — para ayudar a los desarrolladores a crear gran contenido y experiencias. Una única plataforma de hardware también es bueno para los probadores, por lo que es más fácil de probar y abordar cuestiones específicas de software sin tener que preocuparse de dispositivos de modelos específicos, errores, etc.

Si usted tiene su teléfono en la mano y quiere empezar a jugar con el, desarrollando y distribuyendo aplicaciones, o contribuir a la plataforma de Firefox, los siguientes enlaces le llevarán a donde necesita ir:

  • Firefox OS zone: Para crear sus propias builds del sistema operativo Firefox y contribuir a los proyectos de B2G y Gaia.
  • App Center zone: Para la construcción de aplicaciones web de código abierto compatible con el sistema operativo Firefox.
  • Marketplace zone: Para obtener información sobre la publicación y distribución de aplicaciones.
  • Firefox Marketplace: El mejor lugar para encontrar y publicar nuevas aplicaciones del sistema operativo de Firefox.

Si desea obtener más información sobre actualizar el sistema operativo, recuperación, instalación de aplicaciones, o especificaciones del teléfono, usted encontrará la información que necesita en los dos artículos siguientes:

  • Configuración Inicial: Pasos esenciales a seguir para configurar el ordenador para trabajar con su Flame
  • Actualizando su Flame: Cómo actualizar Firefox OS en tu Flame, instalar aplicaciones, y solucionar problemas y probarlo.

Obtener un dispositivo

Las ventas de Flame han llegado a su fin. Todavía hay un montón de oportunidades para los colaboradores de Mozilla para obtener Flame gratuitos, incluyendo una próxima segunda ronda del programa Foxtrot. Si tiene preguntas acerca de cómo obtener un dispositivo gratis para desarrollo o pruebas, por favor, comuniquese con Asa Dotzler o Marcia Knous (marcia) en el canal #fxos en IRC.
