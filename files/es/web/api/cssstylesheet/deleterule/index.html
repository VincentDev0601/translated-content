Resumen

El método deleteRule elimina una regla de la hoja de estilos actual.

Sintaxis

stylesheet.deleteRule(index)

Parámetros

  • index es un numero de tipo long, que indica la posición de la regla.

Ejemplo

 myStyles.deleteRule(0);
