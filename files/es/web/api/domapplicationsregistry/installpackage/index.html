Resumen

Instala una aplicación empaquetada.

Solo disponible en Firefox OS.

Sintaxis

installPackage: function(aURL[, aParams])

Parametros

  • receipts: Será utilizado para especificar los recibos de pago para esta instalación.
  • categories: Será utilizado para especificar las categorías de la aplicación web.

Resultados

aURL
El URL del paquete de aplicación a instalar. Esta debe ser una URL completa y absoluta.
aParams
Opcionalmente, un objeto con información adicional:
DOMRequest
