Summary

parentStyleSheet returns the object type of style sheet, in which the rule is defined.

Syntax

stylesheet = cssRule.parentStyleSheet

Parameters

  • stylesheet stylesheet is an object type.

Instance

if (bgRule.parentStyleSheet! = MySheet) {
   // Style alien rule!
}
