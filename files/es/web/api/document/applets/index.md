### Resumen

`applets` Devuelve una lista ordenada de los
_applets_
del documento.

### Sintaxis

    nodeList = document.applets

### Ejemplo

    // ( Cuando sabes que el segundoapplet es el que quieres )
    my_java_app = document.applets[1];
