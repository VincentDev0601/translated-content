Install.addFile

La méthode addFile de l'objet Install est la manière standard de mettre les fichiers en file d'attente pour l'installation.

var xpiSrc = "file.txt";
initInstall(
  "Ajout d'un fihcier",
  "testFile",
  "1.0.1.7",
  1);
f = getFolder("Program");
setPackageFolder(f);
addFile(xpiSrc);
if (0 == getLastError())
   performInstall();
else
   cancelInstall();
