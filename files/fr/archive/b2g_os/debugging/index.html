Vous pouvez réaliser trois types de débogage sur Firefox OS.

  • Vous pouvez faire un débogage haut-niveau en lançant Gaia (ainsi que de nombreuses applications web compatibles avec B2G) dans Firefox 15 ou plus récent sur ordinateur. Ceci vous permet d'utiliser les excellents outils pour développeur disponibles dans Firefox pour vous aider à déboguer l'interface utilisateur de Gaia, tout comme votre application web.
  • Vous pouvez utiliser l'application dédié B2G pour ordinateur.
  • Ou vous pouvez faire tourner Firefox OS sur votre appareil mobile ou un simulateur sous le contrôle du débogueur gdb. Et il y a des outils pour vous aider à faire tourner le client B2G sur ordinateur.

Cette page fournit les liens vers les informations utiles sur les différents moyens de déboguer votre code Firefox OS.
