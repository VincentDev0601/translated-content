Le LG Fireweb est un smartphone haut de gamme sous Firefox 1.1 avec un appareil photo de 5 mégapixels.

Acheter un appareil

Le téléphone peut être acheté ici :

  • Commander le LG Fireweb sur Movistar Uruguay
  • Commander le LG Fireweb sur Vivo Brésil

Disponibilité

  • Brésil
  • Uruguay

 

Spécifications de l'appareil

Davantage de caractéristiques sont listées sur notre page de spécifications des téléphones et appareils.

  • Chipset : Qualcomm Snapdragon MSM7227A
  • CPU : processeur Cortex A5 à 1 GHz
  • GPU : Adreno 200
  • Photo : 5 MP, 2592 х 1944 pixels, flash LED
  • Batterie : batterie Li-Ion 1540 mAh
  • Affichage : écran tactile capacitif TFT 4,0 pouces
  • Résolution: 320 x 480 (densité de pixels ~144 ppp)
  • Stockage interne : ROM 2 Go
  • Stockage externe : microSD (max. 32 Go)
  • Dimensions : 113,8 x 66,5 x 9 mm
  • Poids : Env. 122 g

Coloris disponible
