Installation de Firefox OS sur votre Nexus 5

Cette section fournit les instructions pour flasher Firefox OS sur votre appareil Nexus 5.

Généralités

Tous les utilisateurs doivent avoir adb et fastboot fonctionnels.  Si vous n'avez pas adb ni fastboot, vous pouvez les télécharger depuis le SDK d'Android dont ils font partie.

Les utilisateurs de Windows doivent également installer les pilotes USB de Google. Téléchargez les pilotes USB, et suivez les instructions d'installation.

Flashage de Firefox OS sous Windows

  1. Téléchargez la version nexus-5-l.zip.
  2. Décompressez le fichier, et vérifiez que flash.bat est présent dans le dossier b2g-distro.
  3. Branchez votre Nexus 5 sur votre ordinateur avec un câble USB.
  4. Vérifiez que l'appareil est allumé, et que le mode adb est activé (Firefox OS : Paramètres > Développeurs > ADB et outils de développement ; Android: Settings > Developer options > USB debugging.)
  5. Double-cliquez sur le fichier flash.bat.

Flashage de Firefox OS sous Mac/Linux

  1. Téléchargez la version nexus-5-l.zip.
  2. Décompressez le fichier, et vérifiez que flash.sh est présent dans le dossier b2g-distro.
  3. Branchez votre Nexus 5 sur votre ordinateur avec un câble USB.
  4. Ouvrez votre terminal et entrez dans le répertoire où se trouve flash.sh — e.g : cd nexus-5-l/b2g-distro.
  5. Entrez la commande adb devices pour vérifier que l'appareil est détecté. (Voir l'aide et les informations de dépannage sur la configuration de la commande adb.)
  6. Entrez la commande chmod +x ./flash.sh pour rendre le script de flashage exécutable.
  7. Entrez la commande ./flash.sh pour procéder à l'installation.

Une fois le processus terminé, votre téléphone redémarrera sous Firefox OS.
