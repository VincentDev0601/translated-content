Spécifications de l'appareil

Logiciel

  • OS : FirefoxOS 1.3T
  • Version Gecko : 28.1
  • Apps intégrées : Agenda, Photo, Horloge, Courriel, Gestionnaire de fichiers, FM, Galerie, Marketplace, Musique, Radio, Paramètres, Suivi conso, Vidéo.
  • Apps du fabricant : Cherry Fun Club, eWarranty.

Matériel

  • Chipset : Spreadtrum SC6821
    • CPU : ARM Cortex-A5
      • Coeurs : 1
      • Fréquence : 1Ghz
    • GPU : ARM Mali-400
  • Mémoire : 128 Mo
  • Stockage interne : 256 Mo
    • 72,8 Mo disponibles pour les applications
    • 8,9 Mo disponibles pour les données multimédia
  • Stockage externe : Carte MicroSD jusqu'à 32 Go
  • Écran :
    • Taille : 3,5 pouces
    • Technologie : TFT, Capacitif
    • Résolution : 320x480 pixels HVGA
    • Densité de pixels : 165 PPP
    • Couleur : Oui
  • Photo :
    • Arrière : 2 MP, focale fixe
    • Frontale : Aucune
  • Batterie :
    • Tension : 3,7V
    • Capacité : 1100mAh/4.,7Wh
    • Amovible: Oui
    • Modèle : CM-8P
    • S/N: CM1410097462
  • Dimensions : 115 x 62,3 x 12,2 mm
  • Poids : Env. 108g
  • Audio : Écouteurs, haut-parleur, jack 3,5mm
  • USB : Micro USB, USB 2.0
  • SIM : Double SIM
  • WLAN :
    • Normes : 802.11b/g/n
    • Securité : WEP, WPA-PSK, WPA-EAP
  • Bluetooth :4.0
  • Réseau : GPRS, EDGE, WAP, MMS
  • Radio FM : Oui
  • GPS : Oui
  • Capteurs : Accéléromètre
