Le Huawei Ascend Y300 II est très semblable au Ascend Y300 basé sur Android, à l'exception qu'il fait tourner Firefox OS 1.1 et non Android, de Google.

Acheter un appareil

Le Huawei Ascend Y300 II est disponible via les canaux de distribution classiques.

Disponibilité

  • Philippines
  • Mexique

Opérateurs​

  • Movistar

Mise à jour du logiciel

L'image système de l'appareil est verrouillée, il n'est donc possible de mettre à jour le logiciel que lorsque l'opérateur téléphonique publie une mise à jour OTA. L'image système de l'appareil est basée sur Firefox OS 1.1.

Spécifications de l'appareil

  • CPU : Qualcomm Snapdragon MSM8225 S4 Play, double cœur 1 GHz, ARMv7 Cortex A5
  • GPU : Qualcomm Adreno 203
  • Caméra: 5 mégapixels avec flash LED, frontale VGA 0,3 mégapixels, Geotagging
  • Batterie : 1730 mAh
  • Affichage : affichage 16 M couleurs 4,0”, Multitouch
  • Résolution : WVGA 480 x 800 pixels, 233 ppp
  • Mémoire interne : RAM 512 Mo, 4 Go stockage interne
  • Mémoire externe : microSD, microSDHC jusqu'à 32 Go
  • Dimensions : 124,5 x 63,8 x 11,2 mm
  • Poids : Env. 130 g

Coloris disponibles

  • Noir anthracite
  • Blanc nacré
