Firefox OS 1.1 est une pré-version de Firefox OS, initialement distribuée auprès des partenaires pour être publiée le 21 octobre 2013. Son composant Gecko est basé sur Firefox 18 (voir Firefox 18 pour les développeurs) tout comme pour Firefox OS 1.0.1, mais avec certaines corrections de bogues supplémentaires et l'ajout d'APIs. Cette page détaille les fonctionnalités pour développeur nouvellement implémentées dans Firefox OS 1.1.

HTML

Spécifique à Firefox OS :

  • <input type="file"> activé (voir bug 832923 pour les détails).

JavaScript

Spécifique à Firefox OS :

  • Lors de la détection du support <video> pour différents formats, HTMLMediaElement.prototype.canPlayType retourne à présent de manière correcte false pour les vidéos h.264.

DOM/API

Spécifique à Firefox OS :

  • API Device Storage : Lors de l'obtention d'un curseur à partir de navigator.getDeviceStorage("sdcard").enumerate, this.done est indéfini (voir bug 902565). Cela est corrigé dans Firefox 1.2. Le bogue référencé aborde de possibles vérifications de compatibilité/contournements pour ce problème.
  • Les APIs relatives aux SMS, MMS et MobileMessage sont arrivées
    • window.navigator.mozMobileMessage
    • window.navigator.mozSms
    • MozMmsEvent
    • MozMmsMessage
    • MozMobileMessageManager
    • MozMobileMessageThread
    • MozSmsEvent
    • MozSmsFilter
    • MozSmsManager
    • MozSmsMessage
    • MozSmsSegmentInfo
    • nsIMobileMessageCallback (API interne à Gecko)
    • nsIMobileMessageCursorCallback (API interne à Gecko)
    • nsIMobileMessageService (API interne à Gecko)
    • nsIMobileMessageDatabaseService (API interne à Gecko)
    • nsIMmsService (API interne à Gecko)
    • nsISmsService (API interne à Gecko)
  • Simple Push (API de notifications Push) est arrivé
  • WAP Push est arrivé

Support HD dans Firefox OS

Depuis Firefox OS 1.1, il existe des versions HD disponibles pour apporter ce support à des appareils avec écrans à haute densité. Il s'agit principalement d'un changement visuel uniquement, traité de la même façon que s'il s'agissait d'autres appareils de résolutions différentes, et rien de plus ne sera visible comme différence au niveau de la plate-forme sous-jacente.

Problèmes corrigés dans Firefox OS

  • 817972 : [Bluetooth][File-Transfer] Support du transfert de plusieurs fichiers.
  • 834672: AppProtocolHandler.js ne devrait pas lever d'exception.
  • 839500 : identity.js, dans Gaia, ouvre un processus navigateur qui reste bloqué en avant-plan. Par conséquent, si vous ouvrez l'application marketplace, vous perdez ~10% de mémoire disponible jusqu'au redémarrage du téléphone.
  • Correction mémoire pour les connexions Persona.
  • 834773: Utiliser l'activité Choisir et valider une image pour la galerie — l'application se bloque sans percevoir de résutat pendant 5–10 secondes

Plantages résolus

Fonctionnalités notables de Firefox OS 1.1

  • Le support des Messages Multimédia (MMS) a été ajouté à l'application des messages ce qui permet d'envoyer des images, vidéos et sons aux contacts, ou un message textuel à plusieurs personnes en même temps.
  • L'enregistrement des images, vidéo et sons depuis le navigateur est maintenant géré.
  • Les contacts des comptes de messagerie peuvent à présent être importés depuis Gmail et Windows Live Mail (Outlook).
  • Le téléchargement des pièces-jointes graphiques, audio et vidéo a été implémenté.
  • Les images de la Galerie peuvent maintenant être mises en pièce-jointe et envoyées.
  • Le mode brouillon pour les courriels a été implémenté.
  • Améliorations du téléphone et des contacts, comme la possibilité d'ajouter facilement un numéro composé à un contact existant ou l'affichage de suggestions pour trouver facilement des contacts.
  • Implémentation de diffusion cellulaire pour la distribution simultanée de messages d'urgence aux utilisateurs abonnés.
  • Améliorations significatives des performances concernant la vitesse du défilement et du lancement des applications.
  • Recherche dans Musique pour trouver de la musique par artiste, album ou titre de chanson.
  • Firefox OS offre des fonctionnalités dans l'Agenda telles que :
    • Vues séparées pour les détails et l'édition.
    • Sélection d'une notification sonore d'alarme.
    • Création d'un événement directement à une date et une heure spécifiques.
