{{CSSRef}}

La valeur initiale d'une propriété CSS est définie par la spécification et varie selon qu'une propriété est héritée ou non.

  • Pour les propriétés héritées, la valeur initiale est uniquement utilisée pour l'élément racine lorsqu'il n'y a pas de valeur définie pour cet élément.
  • Pour les propriétés non-héritées, la valeur initiale est utilisée pour n'importe quel élément lorsque la valeur n'est pas définie pour cet élément.

Le mot-clé {{cssxref("initial")}} a été ajouté en CSS3 afin de permettre aux auteurs d'utiliser cette valeur de façon explicite.
