La propriété __count__ était utilisée pour compter les propriétés énumérables d'un objet mais a été retiréee.

Syntaxe

obj.__count__

Exemples

{ 1: 1 }.__count__              // 1
[].__count__                    // 0
[1].__count__                   // 1
[1, /* trou */, 2, 3].__count__ // 3

Spécifications

Cette propriété n'appartient à aucune spécification.
