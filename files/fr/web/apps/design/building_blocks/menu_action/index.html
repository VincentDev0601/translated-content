Un menu Action présente une liste d'élémenst, liée au contenu de l'application, à partir de laquelle l'utilisateur peut faire une sélection. Consultez le guide de codage pour plus de détails sur la façon de mettre en place un menu d'action dans votre application.

Caractéristiques

  • Ouvert à partir de boutons dans le contenu de l'application, ces boutons sont souvent à l'intérieur des barres d'outils (par exemple, le bouton "Partager" du Navigateur).
  • Les menus d'action contiennent un ou plusieurs éléments.
  • Ces menus se développent en hauteur pour accueillir leurs articles, à un maximum de la hauteur de l'écran. Une fois que la hauteur maximale est atteinte, le contenu défile verticalement. Généralement, la meilleure pratique consiste à essayer de ne comprendre pas plus de cinq éléments, plus un titre de menu.
  • Le titre du menu est optionnel.
  • Le menu se ferme si :
    • Un des élément est sélectionné.
    • Le bouton "annulé" est touché.
