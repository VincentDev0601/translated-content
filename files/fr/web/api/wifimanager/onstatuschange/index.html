Pointe sur un écouteur d'événement pour recevoir l'événement {{event("statuschange")}}. Cet événement se produit lorsque la connexion WiFi change d'état.

Syntaxe

navigator.mozWifiManager.onstatuschange = funcRef

Où funcRef est une fonction à appeler lorsque l'événement {{event("statuschange")}} se produit. Cette fonction de rappel reçoit un objet {{domxref("MozWifiStatusChangeEvent")}} comme premier paramètre.

Spécifications

Ne fait partie d'aucune spécification.
