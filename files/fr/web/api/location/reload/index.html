{{ APIRef("HTML DOM") }}

La méthode Location.reload() recharge la ressource depuis l'URL actuelle.

Si l'action est impossible à cause d'une violation de sécurité, une {{domxref("DOMException")}} de type SECURITY_ERROR est lancée. Cela se produit si l'origine du script appelant est différente de l'origine de la page initialement désignée par l'objet {{domxref("Location")}}, principalement quand le script est hébergé sur un autre domaine. Voir la page Same-origin policy pour plus d'informations.

Syntaxe

object.reload();

Exemples

// Recharge la page actuelle
document.location.reload();
