Résumé

Renvoie la hauteur intérieure d'un élément en pixels. Celle-ci comprend les marges intérieures éventuelles (padding), mais ni la barre de défilement horizontale, ni les bordures ,ni les marges extérieures (margin).

clientHeight peut être calculée comme la propriété CSS height + la propriété CSS padding - la hauteur de la barre de défilement horizontale (si présente).

Syntaxe et valeurs

var h = element.clientHeight;

h est un entier représentant la valeur clientHeight de l'élément element en pixels. clientHeight est une propriété en lecture seule.

Spécifications

clientHeight ne fait partie d'aucune spécification du W3C.

Notes

clientHeight est une propriété non standard, qui a été introduite dans le modèle objet d'Internet Explorer.
