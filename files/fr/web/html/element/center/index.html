L’élément HTML <center> est un élément de type bloc qui contient des paragraphes et d'autres éléments de type bloc ou en ligne. Le contenu entier de cet élément est centré horizontalement au sein de son conteneur parent (généralement l'élément {{HTMLElement("body")}}).

Cet élément a été déprécié en HTML4 (et XHTML 1) au profit de la propriété CSS {{cssxref("text-align")}} qui peut être appliqué à l'élément {{HTMLElement("div")}} ou à un élément {{HTMLElement("p")}}. Pour centrer des blocs, on utilisera d'autres propriétés ({{cssxref("margin-left")}} et {{cssxref("margin-right")}} avec la valeur auto par exemple ou les boîtes flexibles).

Interface DOM

Cet élément implémente l'interface {{domxref("HTMLElement")}}.

Note d'implémentation : Jusqu'à Gecko 1.9.2 inclus, Firefox implémentait l'interface {{domxref("HTMLSpanElement")}} pour cet élément.

Exemples

Premier exemple (HTML)

HTML

<center>
  Ce texte sera centré.
  <p>Ainsi que ce paragraphe.</p>
</center>

Notes

Appliquer {{cssxref("text-align")}}: center sur un élément {{HTMLElement("p")}} ou {{HTMLElement("div")}} centre le contenu de ces éléments, tout en laissant leurs dimensions générales inchangées.
