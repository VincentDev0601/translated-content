L'élément <blockquote> (qui signifie bloc de citation) indique que le texte contenu dans l'élément est une citation longue. Le texte est généralement affiché avec une indentation (voir les notes ci-après). Une URL indiquant la source de la citation peut être donnée grâce à l'attribut cite tandis qu'un texte représentant la source peut être donné via l'élément {{HTMLElement("cite")}}.

Attributs

Comme pour tous les éléments HTML, on peut utiliser les attributs universels.

{{htmlattrdef("cite")}}
Une URL qui désigne la source du document ou du message cité. Cet attribut est prévu pour signaler l'information expliquant le contexte ou la référence de la citation

Notes

  • Pour changer l'indentation de <blockquote>, on utilisera la propriété CSS {{cssxref("margin-left")}} ou {{cssxref("margin-right")}} (ou encore la propriété raccourcie {{cssxref("margin")}})
  • Pour les citations courtes, on pourra utiliser l'élément {{HTMLElement("q")}}.

Exemples

HTML

<blockquote cite="http://developer.mozilla.org">
  <p>Ceci est une citation issue du Mozilla Developer Network.</p>
</blockquote>
