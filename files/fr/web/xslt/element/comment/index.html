L'élément <xsl:comment> écrit un commentaire dans le document de sortie. Il ne doit contenir que du texte.

Syntaxe 

<xsl:comment>
	MODÈLE
</xsl:comment>

Attributs obligatoires

Aucun.

Attributs optionnels

Aucun.

Type

Instruction, apparaît dans un modèle.

Définition

XSLT 1.0, section 7.4.

Support Gecko

Supporté.
