Myk Melez a donné cette présentation de tutoriel sur les extensions de développement comme faisant partie du « Mozilla Labs Challenge ». Cette présentation fournis une introduction rapide au concept d’extension dans le but de vous apprendre les bases sur le développement d’extension.

Cette présentation répond aux questions suivantes :

  • Qu’est ce qu’une extension, que sont elles capable de faire, et de quoi sont elles composé ?
  • Comment configurer son environnement de développement sur Windows, Mac OS X, ou Linux ?
  • Comment faire un extension ? Quels sont les fichiers et dossiers requis ?
  • Comment empaqueter et tester son extension ?
  • Une simple extension est capable d’insérer du code XUL dans la fenêtre du navigateur en se superposant, utilisant les feuilles CSS, le DOM au travers des événements JavaScript, et localisant ses chaînes de caractère avec un DTD et un fichier de propriété.
