Les pages suivantes fournissent diverses techniques pour améliorer l’accessibilité des formulaires web :

  • Indications élémentaires pour les formulaires : ajouter des suggestions et des descriptions pour les champs invalides ou obligatoires
  • Alertes : Utiliser les alertes pour fournir des messages d'erreur de validation côté client
  • Labels multi-options: fournir des labels de formulaire complexes avec un contrôle pour chaque label

Voir également l’article Yahoo! à propos de la validation des formulaires et d’ARIA (en anglais) (version archivée par archive.org), couvrant pour une bonne part le même sujet.
