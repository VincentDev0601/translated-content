{{ApiRef("HTML DOM")}}

送至視窗的 close event 的處理器（handler）。（不支援 Firefox 2 及 Safari）

語法

window.onclose = funcRef;

參數

  • funcRef 為函式的參照。

範例

window.onclose = resetThatServerThing;
