<h2 id="哪邊出錯了？">哪邊出錯了？</h2>

<p>使用 CORS 的 {{Glossary("HTTP")}} 請求失敗，因為 HTTP 連線在網路或協定層級失敗。這個錯誤和 CORS 沒有直接關係，but is a fundamental network error of some kind.</p>
