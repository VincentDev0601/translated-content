escape 和 unescape 函數

escape 和 unescape 函數可讓你把字串編碼或解碼。escape 函數會返回以 ISO Latin 字元集來表示參數的十六進制編碼。unescape 函數會對指定的十六進制編碼值返回 ASCII 字串。

這些函數的語句是︰

escape(string)
unescape(string)

這些函數主要是在伺服端的 JavaScript 用來編碼或解碼 URL 裡的名稱和值的配對。

escape 和 unescape 函數在非 ASCII 字元下無法正確運作，也已被廢棄。在 JavaScript 1.5 及其後版本中，可以改用 encodeURI、decodeURI、encodeURIComponent、decodeURIComponent。
