Firefox 3 加入了兩個新的 events 讓你可以決定 drag 什麼時候開始、什麼時候結束。這些 events 是 HTML 5 規格書的草稿中新加入的。

 

drag
drag operation 開始時送出。
dragend
drag operation 結束時送出。

如想知道更多，請看 [Drag and Drop.]
