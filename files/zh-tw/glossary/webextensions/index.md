<p>WebExtensions 是一種跨瀏覽器系統，用於開發 Firefox 的瀏覽器附加元件。這個系統提供許多 API 能夠在不同的瀏覽器中執行，例如 Mozilla Firefox、Google Chrome、Opera 瀏覽器和 Microsoft Edge。</p>
