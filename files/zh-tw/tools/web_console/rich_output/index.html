當網路主控台印出物件時，它不但會顯示物件名、還會包括該物件資訊的 richer set。尤其是它要：

  • 提供某些類型的額外資訊
  • 開啟物件屬性的詳細檢查
  • 為 DOM 元素提供 richer information，並讓你能在檢查中選擇。

特定類型的 rich output

網路主控台給許多物件類型提供了 rich output，包括：

Object
Array
Date
Promise

New in Firefox 36
RegExp
Window
Document
Element
Event
