網頁主控台能：

  1. 記錄網頁相關的資訊：網路請求、JavaScript、CSS、安全性相關的問題，警告和錯誤、以及頁面運行的 JavaScript 相關問題，警告，錯誤，和參考訊息。
  2. 透過執行頁面文中的 JavaScript 表達式與網頁互動
