A API social é uma nova API para tornar mais fácil para os navegadores web a integração com serviços de mídia social. Uma vez que um provedor de serviços sociais é implementado para o Firefox, torna-se possível para o navegador exibir in-chrome controles de usuário e informações relacionadas a esse serviço. Os artigos a seguir explicam como implementar um provedor de serviço social.

Nota: Grande parte do conteúdo para as subpáginas foi elaborado, mas ainda não foi migrado para este diretório; você pode encontrá-lo temporariamente aqui.
