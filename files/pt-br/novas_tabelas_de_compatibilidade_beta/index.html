Você provavelmete chegou aqui através do link nos avisos beta das nossas novas tabelas de compatibilidade. (Não? Quer ver as novas tabelas? Torne-se um testador beta.)

Agradecemos por nos ajudar a testá-las. Elas são parte de um grande projeto, no qual estamos transformado todos os nossos dados de compatibilidade em JSON estruturado.

As novas tabelas aparecerão nas páginas à medida que os dados forem convertidos para o novo formato.

Editando

Nossos dados de compatibilidade foram movidos para arquivos JSON no repositório browser-compat-data no GitHub.

Para contribuir com os dados de compatibilidade, agora você pode enviar uma pull request ou reportar um erro no repositório.

Como ajudar

Se você encontrar um problema com os dados, por favor reporte o erro no repositório do GitHub.

Se a aparência ou funcionalidade das tabelas não parecem adequadas, clique no botão "Reportar um erro" no menu acima da tabela.

Se você tiver um tempo, agradeceremos se você puder completar nossa pesquisa.
