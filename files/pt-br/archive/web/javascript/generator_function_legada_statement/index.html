A expressão de função generator legada é um recurso específico SpiderMonkey, e será removida em algum ponto. Para usos voltados para o futuro, considere usar a {{jsxref("Statements/function*", "function*")}}.
{{jsSidebar("Statements")}}

O declação de generator function legada declara uma generator function legada com os parâmetros específicos.

Você também pode definir funções usando o construtor {{jsxref("Function")}} com functionBody e pelo menos uma expressão {{jsxref("Operators/yield", "yield")}}, e uma {{jsxref("Operators/Legacy_generator_function", "expressão generator function legada", "", 1)}}.

Sintaxe

function nome([param,[, param,[..., param]]]) {
   [declarações]
}
nome
O nome da função.
param
O nome de um argumento que vai ser passado para a função, uma função pode ter até 255 argumentos.
declarações
As declarações que compõem o corpo da função. Isso pode conter pelo menos uma expressão {{jsxref("Operators/yield", "yield")}}.

Descrição
