Gaia é o front-end do Firefox OS, que contem a funcionalidade de administração do sistema e uma suíte interna de aplicativos distribuídos com os dispositivos Firefox OS. Todo o código fonte do Gaia, mesmo o sistema e os IMEs de teclado - é totalmente desenvolvido em HTML5 (HTML + CSS + JavaScript) e WebAPIs abertas. Esse conjunto de documentos contém informações de como cada um dos aplicativos padrões disponíveis na família Gaia funciona

Categoria de funcionalidades Gaia

Os diferentes aplicativos Gaia podem ser agrupados na lista abaixo.

Nota: Muitas as páginas indicadas nesse documento são na verdade páginas README dentro do repo Gaia no Github. Isso é porque muitos dos aplicativos são produzidos em ciclos de desenvolvimento rápido, o que significa ser muito comum haver atualizações diárias, não fazendo muito sentido manter atualizada a documentação no MDN. Os arquivos README atualizados pela equipe de engenharia é uma fonte mais precisa de informação.

Plataforma

Incluindo o Sistema, Configurações, Bloqueio de tela, scripts de compilação e aplicativos Bluetooth.

Aplicativos da plataforma apps: maiores explicações

Aplicativo System
O aplicativo System é o primeiro aplicativo carregado pelo Gecko durante o procedimento de boot do Firefox OS, e ele lida com inúmeras responsabilidades que são necessárias para o funcionamento geral do sistema não sendo escopo de cada um dos demais aplicativos
Aplicativo Browser
O aplicativo Browser (que não é parte do aplicativo System) fornece as funcionalidades de um navegador onde for necessário — incluindo a navegação em uma página, busca e gerenciamento de favoritos.
Gerenciamento das janelas
A funcionalidade do gerenciamento de janelas do Firefox OS — incluindo o ciclo de vida do aplicativo e suas interações, animações e muito mais — é negociado por uma parte específica do aplicativo System. Esse artigo detalha o gerenciamento das janelas.
Preferências
O Aplicativo de preferências possibilita os usuários do Firefox OS configurar o dispositivo e responder as atividades de entrada (API Web activities com um nome de configure), que possibilita outros aplicativos pular para paineis diferentes dentro do aplicativo Preferências para manipular as configurações requisitadas (por exemplo mostrando o painel de configuração do WiFi quando não houver conexão disponível).

Comunicação

Incluindo os aplicativos Discador, contatos, aplicativos de SMS e de FTU.

Produtividade

Incluindo os aplicativos Email, calendário e relógio.

Aplicativos de produtividade: maiores explicações

Calendário
O aplicativo de calendário interno do Firefox OS.
Relógio
Aplicativo padrão do Firefox OS que inclui as funcionalidades de alarme, temporizador e cronômetro.
Email
O aplicativo de email do Gaia.

Mídia

Incluindo os aplicativos Camera, Galeria, Música e Vídeo e algumas funções relacionadas com mídias como bloqueio a DRM e papeis de parede.

Aplicativos de mídia: maiores explicações

Video
Video é um aplicativo simples que executa arquivos de vídeo armazenados no seu dispositivo.
Camera
Camera possibilita ao usuário do Firefox OS capturar e gerenciar vídeos e fotos da câmera do dispositivo e respondendo Web activities do tipo pick originado de outros aplicativos que querem capturar mídias usando as funcionalidades da câmera.

Outros recursos Gaia

Além dessas funções, há várias outras características importantes, como navegador, tela de início, market place, framework de teste, visualizador de PDF, e gerenciador de aplicativo, que são desenvolvidas em estreita colaboração com o Gaia.

pdf.js
pdf.js um aplicativo de visualização de arquivos PDF baseado em HTML5. Observer que o código é mantido em um repo separado do repo do Gaia.
