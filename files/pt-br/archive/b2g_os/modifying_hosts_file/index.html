Em alguns momentos é necessário editar o arquivo hosts em sua maquina, por causa de um ataque, para bloquear conexões indesejadas, para redirecionar requisições para IPs especificos, entre outros. Esse artigo explica como modificar o arquivo hosts do seu telefone Firefox OS. O arquivo está localizado dentro do diretório /system/etc no Linux, e simplesmente etc/ nas novas versões do Mac OSX.

Preparando seu telefone

No aparelho, vá para Painel de configurações de desenvolvedor e marque:

  • "Depuração Remota" (No Firefox OS 1.4 ou posterior, selecione "ADB e Devtools")
  • "Console Habilitado"

Conecte seu celular no computador.

A partir desse ponto os passos para modificar o arquivo hosts são os mesmos usados para o Android. Os desenvolvedores Android irão facilmente realizar a configuração.

Passos no terminal

Abra a janela do terminal.

  1. Remonte a partição /system do dispositivo para obter as permissões de leitura / escrita
    adb remount
  2. Pegue o arquivo hosts no seu computador
    adb pull /system/etc/hosts /tmp
  3. Modifique o arquivo hosts em /tmp/hosts conforme a necessidade e salve o arquivo, por exemplo:
    127.0.0.1         localhost
ip.to.re.direct   hostName
  4. Envie o arquivo hosts modificado para o dispositivo
    adb push /tmp/hosts /system/etc/hosts

Pronto!

Nota: Quer aprender mais sobre ADB? Consulte a Documentação ADB.
