Geeksphones são dispositivos de desenvolvimento produzidos pela Geeksphone baseada em Madrid, em conjunto com a Telefonica e a Mozilla. O acesso ao hardware permite aos desenvolvedores testarem as funcionalidades do Firefox OS em um ambiente real com uma rede de telefonia móvel e características de hardware como acelerômetro e câmera que não é fácil de testar no simulador do Firefox OS. Este artigo de perguntas frequentes o ajudará a responder questões que você possa ter.

Questões antes da compra

Respondendo as questões que você pode ter antes de comprar seu dispositivo Geeksphone.

A quem é destinado os aparelhos da Geeksphone? 

Desenvolvedores são o principal alvo dos Geeksphones. Não são dispositivos para serem utilizados pelo público em geral; foram desenhados apenas para testes e experimentações.

Como adquirir um dispositivo Geeksphone? Quanto custa?

Você pode comprar diretamente na loja virtual do Geeksphone. Lá você encontra os preços. See the Geeksphone store for pricing information. Além disso, em algumas situações a Mozilla disponibilizará dispositivos como parte dos esforços de divulgação.

Os Geeksphones são completamente desbloqueados?

Sim!

Em quais países os Geekphones estarão disponíveis?

Em todos os países. Entretanto, o celular passará pela alfândega do país de destino. Se você tiver qualquer dúvida, contate a alfândega local.

 

Os Geeksphones funcionam com qualquer SIM card?

Sim.

Em quais países os Geeksphones estão disponíveis?

O Geeksphone pode ser enviado para todos os países. Porém depende das regras alfandegárias do país de destino. Se tiver alguma dúvida, entre em contato com a alfândega do seu país.

Qual Geeksphone eu devo comprar?

Atualmente o único Geeksphone disponível é o Revolution - um telefone que vem com o Android, mas que pode ser reformatado com FirefoxOS.

O Peak tem um processador mais rápido, uma câmera melhor e uma tela com a resolução melhor que o Keon. Por isso, muitos desenvolvedores preferem o Peak em relação ao Keon. No entanto, o suporte para dispositivos de alta resolução ainda está em implementação, portanto, pode haver várias falhas visuais e bugs. O Peak pode ser um dispositivo apropriado se você está disposto a lidar com esses problemas. Uma outra boa escolha se você (i) está interessado em contribuir com o Firefox OS, particularmente na implementação do suporte à alta resolução, (ii) é um desenvolvedor de aplicativos que gostaria de trabalhar com dispositivos high-end que serão vendidos no futuro.

Firefox OS nos Geeksphones

Aqui você encontrará respostas às perguntas sobre como conseguir as imagens do Firefox OS para o Geeksphone, bem como utilizar e atualizar os dispositivos.

Onde posso encontrar estruturas para Geeksphones?

Um índice de estruturas para os dispositivos do Geeksphone estão disponíveis no  Geeksphone website.

Onde posso encontrar arquivos de manifesto das imagens?

Os arquivos de manifesto para todas as imagens estão disponíveis aqui: http://www.geeksphone.com/manifests/.

Como fazer o flash de uma imagem para o meu Geeksphone?

Geeksphone fornece algumas instruções para o flashing de imagens para o seu dispositivo. As imagens fornecidas pela Geeksphone inclui os binários necessários para o flashing no Linux e Windows. Se você estiver usando um Mac, você deve instalar o Android SDK. Se estiver usando o Windows, você precisará instalar os drivers USB.

Como ter suporte para meu Geeksphone?

Para problemas no hardware ou com as imagens, visite o forum da Geeksphone. Você também pode enviar um email para info@geeksphone.com ou telefonar para (+34) 915 51 62 91.

Para questões sobre desenvolvimento de aplicativos para Firefox OS, veja os artigos do MDN em Criando aplicativos para Firefox OS bem como no Developer Hub na Firefox Marketplace.

Para questões sobre o uso do using Firefox OS, visite SUMO.
