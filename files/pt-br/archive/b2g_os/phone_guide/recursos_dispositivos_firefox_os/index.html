Botões e Controles

Um típico dispositivo Firefox OS possui um pequeno número de botões:

Botão "Home"
Esse botão geralmente fica localizado abaixo da tela, centralizado. Pressionando-o você retorna à tela dos aplicativos. Ao pressionar e segurar, será aberta uma visão do tipo "cartas", apresentando os aplicativos abertos. Para finalizá-los, basta movê-los para cima.
Botões de Volume
Na lateral esquerda, você encontra o botão de volume. Pressione na parte superior para aumentar o volume e na inferior para abaixar o volume.
Botão "Power"
Fica localizado no canto superior direito do dispositivo.

Requisitos mínimos de Hardware

Os dispositivos baseados no processador ARM normalmente são robustos o suficiente para suportar o Firefox OS. Esse artigo cobre os requisitos básicos de hardware bem como as características recomendadas.

Componente Mínimo Recomendado
CPU ARMv6 Classe Cortex A5 ou superior
ARMv7a com NEON
GPU — Classe Adreno 200 ou superior
RAM 256 MB 512 MB
Conectividade — WiFi
3G
Sensores — Acelerômetro
Proximidade
Luz Ambiente
A-GPS

Nota: Para uma boa experiência do usuário é uma boa ideia escolhere um dispositivo que ofereça um perfil de cores uniforme (implementado pelo drive gráfico) e um fone de ouvido que possua botão para implementar a funcioinalidade "Mudo" e parar e iniciar a execução de música/vídeo. Essas características são comuns nos smartphones modernos.
