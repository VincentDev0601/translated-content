O componente Gecko do Firefox OS 2.1 é baseado no Firefox 33/34. Esta página detalha os recursos para desenvolvedores recentemente implementadas no Gecko que são específicos para o Firefox OS 2.1.

Nota: Para características gerais do Gecko, que estão geralmente também presentes no Firefox OS Gecko, consulte as notas de lançamento do Firefox 33 para desenvolvedores e as notas de versão para desenvolvedores Firefox 34.

Mudanças da plataforma

O Firefox OS costumava ter um "único app por origem" em sua política de segurança, mas esta foi elevada a partir do Firefox 34/Firefox OS 2.1 (leia esta entrada FAQ para mais informações). Caso você ainda precise de suporte para versões anteriores, considere hospedar seus aplicativos em origens distintas; uma estratégia é criar diferentes subdomínios para os seus apps.

Mudanças no Web API

  • Os manipuladores de eventos de emparelhamento do Web Bluetooth API — BluetoothAdapter.ondevicepaired e BluetoothAdapter.ondeviceunpaired — foram adicionados (veja bug 1036234).
  • Métodos de emparelhamento atualizados no Web Bluetooth API foram adicionados (veja bug 1036233):
    • Promise<void> BluetoothAdapter.pair(DOMString aAddress)
    • Promise<void> BluetoothAdapter.unpair(DOMString aAddress)
    • sequence<BluetoothDevice> BluetoothAdapter.getPairedDevices
  • O suporte para o Web Speech API está pendente: um patch foi submetido (see bug 1032964).
  • O FileHandle API foi renomeado para MutableFile, e o LockedFile renomeado para FileHandle (bug 1006485).
