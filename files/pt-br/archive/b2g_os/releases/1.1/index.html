Firefox OS 1.1 é uma versão de pré-lançamento do Firefox OS, oferecida pela primeira vez aos parceiros para lançamento em 21 de outubro de 2013 componente O Gecko é baseado no Firefox 18 (ver Firefox 18 para desenvolvedores) como Firefox OS 1.0.1 , mas com algumas correções de bugs e APIs adicionais acrescentou. Esta página detalha os recursos de desenvolvedor recém-implementados no Firefox OS 1.1.

HTML

Firefox OS-específica:

  • <Input type = "file"> habilitado (consulte bug 832923 para obter detalhes).

Apoio Firefox OS HD

Desde Firefox OS 1.1, há versões HD disponíveis para fornecer suporte HD para dispositivos de alta DPI. Isto é mais apenas uma mudança visual, tratados da mesma forma que você lidaria com outros dispositivos de diferentes resolução, e você vai encontrar nada muito em termos de diferenças entre as plataformas subjacentes.

Falhas resolvidas

Destaques de critério para o Firefox OS 1.1

  • Mensagens suporte multimídia (MMS) adicionado ao aplicativo de mensagens para que você possa enviar fotos, vídeo e áudio para contatos ou uma mensagem de texto para várias pessoas ao mesmo tempo.
  • Guardar imagens, vídeo e áudio a partir do navegador é agora suportado.
  • Conta de e-mail de contatos agora pode ser importado a partir do Gmail e Windows Live Mail (Outlook).
  • Email imagem, áudio e vídeo anexado download foi implementada.
  • Anexando e envio de imagens Galeria agora suportado.
  • Modo de rascunho e-mail foi implementado.
  • Melhorias para o discador e contatos, como adicionar facilmente um número discado para um contacto e sugestões discador existentes para encontrar facilmente os contactos.
  • Cell Broadcast implementado para entrega simultânea de mensagens de emergência para usuários inscritos.
  • Principais melhorias de desempenho em torno do tempo de inicialização do aplicativo e rolagem.
  • Música de busca para encontrar músicas por artista, álbum ou título da música.
  • Firefox OS oferece Calendário características tais como:
    • Vistas de detalhe e de edição separada.
    • Seleção de notificação som do alarme.
    • Criação evento direto na data / hora específica.
