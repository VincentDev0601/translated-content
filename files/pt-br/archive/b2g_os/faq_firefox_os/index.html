O que é o Firefox OS?
É um novo sistema operacional móvel construído inteiramente utilizando os padrões da web aberta. Ele permite que todas as funções do dispositivo (chamadas, mensagens, navegação etc.) sejam desenvolvidas como uma aplicação HTML 5 que pode acessar os recursos subjacentes de um telefone (disponível somente para aplicações nativas em outros ecossistemas).
Como faço para obter um telefone para desenvolvedores?
Você pode comprar o telefone de referência para desenvolvedor Flame no site everbuying.
Se eu não quiser comprar um telefone, eu ainda posso desenvolver um app para Firefox OS?
Claro! Você pode testar seu app no Android (usando Marketplace para Firefox) ou em seu desktop usando o Firefox OS simulator.
Onde posso baixar o Firefox OS para testar em meu próprio telefone?
Veja Compilando e Instalando o Firefox OS.
Como posso testar meu aplicativo no Firefox OS?
No Android ou usando o Firefox OS simulator.
Qual é a lista completa de APIs do dispositivo? Você planejam padronizá-las?
Uma lista completa pode ser encontrada em https://wiki.mozilla.org/WebAPI#APIs. Padronização está em andamento.
Suas APIs do dispositivo estão sendo padronizadas para uso multiplataforma?
Sim, as APIs são o resultado de trabalho com diversos parceiros e vendedores e alguns deles já acabaram em outras plataformas. Quase toda compania de tecnologia tem o problema ao ativar o acesso ao hardware por Web Apps que o usuário final tem e as nossas implementações são um bom ponto de partida para fazer isso acontecer para outras plataformas.
