Gaia é a interface de usuário do Firefox OS e uma suíte de aplicativos padrão: inclui o bloqueio de tela, tela inicial, discador e outras aplicações. Essencialmente, o Gaia é um conjunto de aplicativos web complexos que são executados no topo da plataforma do Firefox OS. Esse conjunto de artigos cobre tudo o que você precisa saber para contribuir para o projeto Gaia.

Neste guia nós levaremos você a um fluxo de trabalho para contribuir com o Gaia — o que significa adicionar recursos no código fonte do Gaia e trabalhar para resolver bugs. O primeiro conjunto de artigos podem ser lidos em ordem, ou você pode ir direto a uma seção relevante se você precisar rever um aspecto específico do processo.

Depois disso, nós fornecemos material de referência e informação cobrindo tópicos adicionais.
