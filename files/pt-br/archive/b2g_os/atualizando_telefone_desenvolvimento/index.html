Nota: Existe um guia mais detalhado para atualizar telefones de desenvolvimento — incluindo o Geeksphone. Você deve ler Atualizando seu telefone Firefox OS / Geeksphone.

Se você tem um telefone para desenvolvimento da Geeksphone e gostaria de atualizar seu binário, siga os passos a seguir. Só são válidos para os modelos da GeeksPhone Peak e Keon.

  1. Certifique-se que você tem ao menos 50% de carga da bateria. Isso é importante, pois se a bateria do seu telefone acabar durante o processo de atualização, você poderá ter que reiniciar todo o processo.
  2. Com o telefone desconectado da USB selecione a opção Depuração remota.
  3. Baixe o Driver USB e descompacte-o. Lembre o diretório para o qual você descompactou, pois você pode necessitar no próximo passo.
  4. Use um cabo USB para contectá-lo ao seu computador. Se estiver usando Windows, será necessário instalar alguns drivers. Se não forem encontrados pelo Windows Update, você pode indicar o diretório no qual você descompactou o driver no passo anterior.
  5. Baixe a imagem apropriada para o seu telefone (Keon ou Peak). Todas as imagens estão disponíveis na página de Download da GeeksPhone.
  6. Descompacte o arquivo num diretório, vá para o diretório descompactado e execute o script: Flash.sh para Linux, Flash.bat is para Windows e flash_mac.sh para Machintosh (você vai precisar do Android SDK instalado).
  7. Espere até o dispositivo ser reiniciado e mostrar a configuração de primeiro uso.


Se você encontrar algum problema, coloque suas dúvidas no fórum da Geeksphone.
