Aplicativos Firefox OS nada mais são do que Open Web apps (inglês) que estão instalados em um telefone rodando o Firefox OS. Isso significa que você pode seu conhecimento no desenvolvimento de front-ends para criar aplicações incríveis para serem executadas em telefones Firefox OS.

Para aprender mais sobre as técnicas envolvidas na construção de aplicativos ao invés de Web sites, estude nosso tutorial: Começando a construir aplicativos. O desenvolvimento de aplicativos para um Sistema Operacional específico tem suas ressalvas. Você pode achar informações relevantes no noss guia rápido: Escrevendo aplicações web para o Firefox OS

Uma  vez familiarizado com a base, você poderá ler Referência de Desenvolvimento de API e para uma informação básica sobre desenvolvimento de aplicativos Firefox OS, veja Firefox Marketplace Developer Hub.

Nota: Firefox OS atualmente é identifica usando a mesma string UA que o Firefox para Android, porém sem a cláusula Android. Por exemplo: Mozilla/5.0 (Mobile; rv:18.0) Gecko/18.0 Firefox/18.0.
