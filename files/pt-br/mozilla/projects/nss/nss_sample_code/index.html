Código NSS

As seções seguintes mostram como NSS pode ser utilizada para operações criptográficas, gerenciamento de certificados, SSL, etc. Apresenta também, boas práticas em aplicações de criptografia.

  1. Código fonte 1: Geração de chave e transporte entre servidores
  2. Código fonte 2: Encriptação simétrica
  3. Código fonte 3: Hashing, MAC
  4. Código fonte 4: Encriptação PKI
  5. Código fonte 5: Encriptação PKI com chave pública private em formato DER
  6. Código fonte 6: Chaves simétricas persistentes em banco de dados NSS
