Zonas MDN provê à você em um único lugar acesso à informação sobre áreas especificas ou produtos. Aqui está uma lista de todas as zonas disponíveis pra você.

Desenvolvimento Web e Aplicativos

Central de Aplicativos
Aprenda como criar aplicativos Web abertos — ricas experiências que executam em múltiplos dispositivos e fatores de forma —usando o mesmo padrão Web e tecnologias abertas que você já conhece.
Área de Aprendizado
Esta zona contém todos artigos para ajudar recém-chegados para o desenvolvimento Web iniciarem seu excitante mundo novo.
Ferramentas de Desenvolvimento
Aprenda como usar as ferramentas de desenvolvimento Firefox para debugar, testar, e otimizar suas Web apps e sites.
Firefox Marketplace
Aberto, não-proprietário, online marketplace (mercado) para aplicativos Web construídos usando HTML, CSS, and JavaScript. Envie aplicativos para o Firefox Marketplace ou use o código para construir seu próprio marketplace.
Desenvolvimento de Jogos
Aprenda como desenvolver games para Web, como porta de jogos para as tecnologias da Web existente, e como transformar seus jogos em aplicativos web.

Produtos e projetos

Emscripten
Um LLVM (coleção modular e compiladores reusáveis) para o compilador JavaScript; este deixa você compilar, por exemplo, código C++ dentro do código JavaScript que pode ser executado em qualquer navegador.
L20n
Uma estrutura de trabalho (framework) de localização para desencadear a energia da linguagem natural com um código simples.
O Projeto MDN
A Rede de Desenvolvedores da Mozilla (este site) confia em sua comunidade de leitores e contribuidores para crescer e aprimorar. Você pode aprender aqui, como ajudar a usar, contribuir, e construir o código por traz do MDN ! 1
Persona
Um simples novo, privado e único sistema desenvolvido pela Mozilla que permite aos usuários logar em seu Web site usando o email deles, libertando-o do gerenciamento de senhas.

Tecnologias Mozilla

Add-ons
Aprenda como construir e instalar extensões, temas, e plug-ins para em programas baseados na Mozilla, incluindo o popular Firefox Web browser.
Firefox
Aprenda tudo sobre Firefox, como construir, revisar e como construir add-ons especificamente para ele.
Firefox OS
Um novo sistema operacional móvel desenvolvido pela Mozilla ao qual deixa os usuários instalarem e executarem Aplicativos Web Abertos, criados usando HTML, CSS, e JavaScript.
