쿠키 모두 없애기

다음 코드는 Firefox/Seamonkey에서 쿠키를 모두 없애는 법을 보여줍니다.

Components.classes["@mozilla.org/cookiemanager;1"]
          .getService(Components.interfaces.nsICookieManager).removeAll();
