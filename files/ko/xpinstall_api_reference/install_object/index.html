개요

The Install object 는 설치 스크립트들에서 우선적으로 사용되어 집니다. 어떤 경우든, Install object는 맹목적이고--일반적인 웹 페이지 스크립트의 window object 같이-- object methods를 앞에 쓸 필요도 없습니다. 예를 들어 아래 두줄은 동등한 결과를 나타냅니다.

f = getFolder("Program");
f = Install.getFolder("Program");

설치 스크립트는 Install object를 호출함으로써 구성되어지며, 일반적으로 아래와 같은 형식을 가집니다.

설치 초기화 하기
설치를 위한 이름, 필요한 레지스트리, 그리고 버젼 정보로 initInstall를 호출하세요.
설치할 파일을 추가하기
getFolder를 호출하여 file objects를 얻고, addFile를 필요한 만큼 호출하여 그것들의 object refs를 넘겨주어 파일들을 추가합니다.
설치를 실행하기
성공적으로 파일들이 추가되었는지 확인하시고 (e.g., 주 설치 메소드들로부터 Return Codes 코드를 확인), 만약 모든 것들이 정상적이라면 설치를 진행하십시요.
performOrCancel();
function performOrCancel()
{
  if (0 == getLastError())
     performInstall();
   else
     cancelInstall();
}

완벽한 스크립트 예제를 원한다면, Script Examples를 참조하세요..
