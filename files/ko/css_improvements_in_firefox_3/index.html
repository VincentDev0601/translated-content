{{ Fx_minversion_header(3) }}

Firefox 3에서는 CSS 구현에 있어 수 많은 향상을 제공합니다. 이러한 변화들의 상세 내용은 해당 문서에 문서화되어 있으므로, 여기서는 이러한 내용들에 대해 개략적으로 언급하고 어떤 변화가 있는지 개발자들이 쉽게 배울 수 있도록 하겠습니다.

  • {{ Cssxref("display") }} 프로퍼티의 inline-block과 inline-table 값이 구현되었습니다.
  • 모든 플랫폼에서 {{ Cssxref("font-size-adjust") }} 프로퍼티가 동작합니다. 이전에는 Windows에서만 지원되었습니다.
  • {{ Cssxref("color") }}의 값으로 rgba()와 hsla()를 사용할 수 있습니다. ({{ Bug(147017) }})
  • {{ Cssxref(":default") }} 의사 클래스를 지원합니다. ({{ Bug(302186) }})
  • {{ Cssxref("width") }}, {{ Cssxref("min-width") }}, {{ Cssxref("max-width") }} 프로퍼티의 값으로 -moz-max-content, -moz-min-content, -moz-fit-content, -moz-available 들을 사용할 수 있습니다. ({{ Bug(311415) }}와 {{ Bug(402706) }})
  • HTML soft hyphens (&shy;)이 지원됩니다.
  • Pre-formatted 탭이 가변글꼴에서 CSS 2.1 스펙에 따라 더 잘 동작합니다.
  • {{ Cssxref("ime-mode") }} 프로퍼티가 지원됩니다.
  • CSS text-rendering 프로퍼티가 지원됩니다. ({{ Bug(387969) }})
  • -moz-border-*-start와 -moz-border-*-end CSS 프로터피가 구현되었습니다 ({{ Bug(74880) }}).
  • -moz-initial이 대부분의 CSS 프로퍼티에 구현되었습니다. (quotes와 -moz-border-*-colors 제외) {{ Bug(80887) }}.
  • 모든 CSS 프로퍼티에 대해 window.getComputedStyle()을 지원합니다 ({{ Bug(316981) }}).
  • {{ Cssxref("content") }}에 none 값을 사용할 수 있습니다 ({{ Bug(378535) }}).
  • {{ Cssxref("cursor") }}에 none 값을 사용할 수 있습니다 ({{ Bug(346690) }}).
  • 배경 이미지를 -moz-border-radius로 정확히 자를(clip) 수 있습니다 ({{ Bug(24998) }}).
  • {{ Cssxref("white-space") }}의 pre-wrap 값을 지원합니다 ({{ Bug(261081) }}).
  • {{cssxref(":first-child")}}, {{cssxref(":only-child")}}, {{cssxref(":last-child")}} ({{ Bug(73586) }}), {{cssxref(":empty")}} ({{ Bug(98997) }}) 등의 선택자(selector)와 + 조합(combinator) ({{ Bug(229915) }})이 동적으로 업데이트됩니다 ({{ Bug(401291) }}).
  • {{ Cssxref("z-index") }}의 음수값을 정확히 지원합니다.
