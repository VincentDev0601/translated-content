휴대전화의 핵심 기능 중 하나는 SMS 메시지 전송과 수신이다. 이 기능은 WebSMS API를 사용하여 구현 가능하다. 다음은 구현 방법 예제이다.

// SMS object
var sms = navigator.mozSms;

// Send a message
sms.send("123456789", "Hello world!");

// Receive a message
sms.onreceived = function (event) {
    // Read message
    console.log(event.message);
};
