예제

아래 코드는 마지막으로 DOM 이 변경된 시간을 페이지의 title 에 표시합니다.

해당 이벤트 핸들러 안에서 DOM 을 변경할 경우 무한 루프에 빠지기 쉬우므로 매우 신중해야 합니다.
document.body.addEventListener('DOMSubtreeModified', function () {
  document.title = 'DOM Changed at ' + new Date();
}, false);
