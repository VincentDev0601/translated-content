last 함수는 식 평가 문맥(expression evaluation context)과 크기가 같은 숫자를 반환합니다.

구문

last()

반환

식 평가 문맥과 크기가 같은 정수.

주의

  • 이 함수는 특정 노드가 노드 집합의 마지막 노드인지를 결정하기 위해 position() 함수와 함께 자주 씁니다.
