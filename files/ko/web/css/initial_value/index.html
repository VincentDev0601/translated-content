{{cssref}}

CSS 속성의 초깃값은 속성의 기본값으로, 속성 정의 표가 명시하고 있습니다. 초깃값의 사용처는 상속 여부에 따라 다릅니다.

  • 상속 속성의 경우, 지정값이 없는 이상 초깃값은 루트 요소에만 쓰입니다.
  • 비상속 속성의 경우, 지정값이 없는 이상 초깃값은 모든 요소에 쓰입니다.

참고: {{cssxref("initial")}} 키워드를 사용해 명시적으로 초깃값을 지정할 수 있습니다.
