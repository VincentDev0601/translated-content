XForms Essentials
XForms로 안내 여행을 제공하는 온라인 책
XForms는 W3C가 계획한 대로 온라인 폼의 미래입니다. XML Schema, XPath, XML Events과 같은 W3C의 다른 표준을 끌어다가, XForms는 현재 HTML 폼 모델에 있는 한계를 처리하려 합니다. XForms가 표에 가져다주는 다른 강점은 겉모양에서 데이터 분리, 엄격한 데이터 형 검사, 이름/값 쌍 대신 서버에 XML 데이터를 제출(submit)하는 능력, 넓고 다양한 장치에 표시할 수 있도록 폼을 작성하는 기술법입니다. XForms는 W3C specification입니다.

XForms 지원은 Mozilla XForms Extension을 설치하여 Firefox와 SeaMonkey에 추가할 수 있습니다.
