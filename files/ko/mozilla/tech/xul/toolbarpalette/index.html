Firefox only

toolbar에 넣을 수 있는 항목들의 팔레트입니다. 이 요소는 직접 출력되지 않으며, 대신 도구 모음 설정 대화창의 항목들로 출력됩니다. toolbarpalette의 자식 요소들은 툴바에 포함될 수 있는 toolbarbutton과 toolbaritem 이어야 합니다. 여백 항목은 자동적으로 포함되기 때문에 직접 추가하면 안됩니다.

id가 BrowserToolbarPalette인 toolbarpalette를 overlay내에 포함하여 Firefox 브라우저에 사용자 정의 버튼을 추가할 수 있습니다.
