Cross-Platform Install (XPInstall)는 Mozilla 어플리케이션 수트, Mozilla Firefox, Mozilla Thunderbird, 다른 XUL에 기반을 둔 어플리케이션이 확장기능을 설치하기 위해 쓰는 기술입니다. XPI ("zippy"로 발음하고 XPInstall에서 파생된) 인스톨러 모듈은 파일 루트에 설치 스크립트나 매니페스트(install.js나 install.rdf로 이름 붙인)를 포함하는 ZIP 파일입니다.

XPInstall API Reference
Learn XPI Installer Scripting by Example
이 문서는 XPI 설치 일반을 논의하기 위한 근거로 browser.xpi 설치 패키지의 인스톨러 스크립트를 씁니다.
How to add an installer module to the Windows installer
인스톨러 모듈은 XPI의 루트에 설치 스크립트(install.js로 이름 붙인)를 포함하는 Phil Katz zip format으로 된 패키지입니다. 패키지는 파일과 파일/디렉토리 추가, 이전 혹은 안 쓰이는(obsolete) 파일/디렉토리 삭제, 명령줄 도구 실행들을 포함하는 설치 도중 취하는 동작을 위한 자바스크립트 지시어(directive)를 포함하는 설치 스크립트를 포함합니다.
Creating XPI Installer Modules
이 문서는 Mozilla의 패키징 계획(scheme)을 설명하고 재배포와 설치가 가능하고 사용자가 쓸 수 있는 새 패키지를 만드는 법을 다루는 입문서를 제공합니다.
